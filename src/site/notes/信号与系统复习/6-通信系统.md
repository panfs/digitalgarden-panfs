## 六、通信系统

### 1、幅度调制（AM）基本模型

#### （1）调制基本模型

**通用模型**：
$$y(t) = x(t) \cdot c(t)$$

其中：
- $x(t)$ 为信息（调制）信号
- $c(t)$ 为载波信号
- $y(t)$ 为已调信号

#### （2）调制的作用

- **频谱搬移**：将信号频谱搬移到较高的载波频率
- **频分复用**：允许许多个信号在同一信道中传输
- **天线匹配**：使信号波长适合天线尺寸

#### 【练习题】

**题目71**：已知调制信号 $x(t) = \cos(2\pi \cdot 100t)$，载波 $c(t) = \cos(2\pi \cdot 1000t)$，求已调信号 $y(t)$ 的频谱。

**答案**：
$$y(t) = \cos(2\pi \cdot 100t) \cdot \cos(2\pi \cdot 1000t)$$

利用欧拉公式：
$$= \frac{1}{2}[\cos(2\pi \cdot 1100t) + \cos(2\pi \cdot 900t)]$$

频谱包含两个频率分量：1100 Hz 和 900 Hz。

### 2、正弦载波调制与解调

#### （1）调制

**复指数载波**：
$$y(t) = x(t)e^{j\omega_c t}$$

**频谱**：
$$Y(j\omega) = X(j(\omega - \omega_c))$$

频谱是调制信号频谱的平移，不产生额外副本。

**正弦载波**：
$$y(t) = x(t)\cos(\omega_c t)$$

利用欧拉公式：$\cos(\omega_c t) = \frac{1}{2}(e^{j\omega_c t} + e^{-j\omega_c t})$

**频谱**：
$$Y(j\omega) = \frac{1}{2}X(j(\omega - \omega_c)) + \frac{1}{2}X(j(\omega + \omega_c))$$

**带宽**：$2\omega_m$，其中 $\omega_m$ 为调制信号的最高频率。

**无失真调制条件**：$\omega_c > \omega_m$

#### （2）解调

**同步（相干）解调**：

假设已调信号为：
$$y_m(t) = x(t)\cos(\omega_c t)$$

其中 $x(t)$ 是原始信号（基带信号）。

解调过程：
$$y(t) = y_m(t)\cos(\omega_c t)$$
$$= x(t)\cos(\omega_c t) \cdot \cos(\omega_c t)$$
$$= x(t)\cos^2(\omega_c t)$$
$$= x(t) \cdot \frac{1}{2}(1 + \cos(2\omega_c t))$$

通过低通滤波器（截止频率在 $\omega_m$ 和 $2\omega_c - \omega_m$ 之间）：
$$y_{LP}(t) = \frac{1}{2}x(t)$$

**要求**：发射端与接收端的载波严格同步（同频同相）。

#### 【练习题】

**题目72**：已知调制信号 $x(t)$ 的带宽为 $\omega_m = 10\pi$ rad/s，载波频率为 $\omega_c = 100\pi$ rad/s，求AM信号的带宽。

**答案**：
AM信号带宽：
$$B = 2\omega_m = 2 \times 10\pi = 20\pi \text{ rad/s}$$
$$= 10 \text{ Hz}$$

### 3、单边带调制与频分复用

#### （1）单边带调制 Synchronization Signal Block（SSB）

**目的**：在正弦载波调制中，只传输上边带或下边带，以节省一半传输带宽。

**方法**：
- **上边带**：在常规幅度调制后，通过一个理想高通滤波器（保留 $\omega_c < \omega < \omega_c + \omega_m$）
- **下边带**：在常规幅度调制后，通过一个理想低通滤波器（保留 $\omega_c - \omega_m < \omega < \omega_c$）

**带宽**：$\omega_m$（是DSB-SC的一半）

#### （2）频分复用 Frequency-Division Multiplexing（FDM）

**概念**：将多个不同信源（频谱重叠）的信号调制到不同频率的载波上，使它们的频谱不重叠，从而可以在同一个宽频带信道中同时传输。

**接收端**：
1. 通过带通滤波器分离出各信道信号
2. 分别进行解调恢复原信号

#### 【练习题】

**题目73**：有3个信号，带宽分别为 5 kHz、10 kHz、8 kHz，要进行频分复用传输。若载波频率分别为 100 kHz、120 kHz、140 kHz，求所需的总带宽。

**答案**：
信号1：上边带 100-105 kHz，下边带 95-100 kHz
信号2：上边带 120-130 kHz，下边带 110-120 kHz
信号3：上边带 140-148 kHz，下边带 132-140 kHz

若使用上边带调制，总带宽为：148 kHz - 100 kHz = 48 kHz

### 4、脉冲串载波调制与脉冲幅度调制

#### （1）脉冲幅度调制 Pulse-amplitude Multiplexing（PAM）

**定义**：使用周期脉冲串作为载波，其脉冲幅度随调制信号 $x(t)$ 的样值变化。

$$y(t) = \sum_{n=-\infty}^{+\infty}x(nT_s)p(t-nT_s)$$

其中 $p(t)$ 为脉冲波形。

**与采样的关系**：可看作是零阶保持采样的结果。

#### （2）码间串扰 Inter-symbol Interference（ISI）

**成因**：在PAM中，若信道为低通特性，会使得接收端的脉冲在时域上展宽，相邻脉冲的波形相互重叠，产生**码间串扰**。

#### （3）对抗ISI的方法

**方法1：满足奈奎斯特准则**
根据**奈奎斯特第一准则**，若整个系统（发送滤波器 + 信道 + 接收滤波器）的总频率响应为理想低通滤波器（截止频率为符号率的一半），则可以实现无 ISI 传输。

**方法2：使用带宽受限的脉冲**
- **sinc 脉冲成形**：理想低通滤波器的时域响应是 sinc 脉冲，在发送端使用 sinc 脉冲成形是满足奈奎斯特准则、实现无 ISI 传输的一种方法
- **升余弦脉冲**：常用带限脉冲，可在零点处无码间串扰

---

### 5、时分复用与角度调制

#### （1）时分复用 Time-division Multiplexing（TDM）

**概念**：将多个信号的PAM脉冲在时间上交错排列，以在单一信道上传输。

**系统框图**：
```
信号1 → 采样 → \
信号2 → 采样 →  → 交错排列 → 信道 → 分离 → 低通滤波 → 输出信号
信号3 → 采样 → /
```

**限制**：
- 受到采样定理约束：$T \leq \frac{\pi}{\omega_m}$
- 受脉冲宽度 $\Delta$ 限制（影响所需带宽）

**复用信号数量**：
$$N \leq \frac{T}{\Delta}$$

**系统最小带宽**：
$$B \approx \frac{1}{\Delta}$$

该公式基于时域与频域的对偶关系。矩形脉冲在频域的谱为 sinc 函数，其第一零点位于 $f = 1/\Delta$。要无失真传输宽度为 $\Delta$ 的脉冲，系统带宽至少应覆盖至第一零点。

#### （2）角度调制简介

**基本概念**：通过调制信号改变载波的相位或频率。

**相位调制（PM）**：
$$\theta(t) = \omega_c t + k_p x(t)$$

**频率调制（FM）**：
瞬时频率：$\omega_i(t) = \omega_c + k_f x(t)$

### 6、调制方式优缺点对比

| 调制方式 | 带宽 | 优点 | 缺点 | 应用场景 |
|---------|------|------|------|---------|
| **AM (标准幅度调制)** | $2\omega_m$ | • 接收端简单，可用包络检波（无需载波同步）<br>• 技术成熟，成本低 | • 载波功率浪费，功率效率低<br>• 抗噪声性能较差<br>• 带宽相对较宽 | • 广播电台（AM广播）<br>• 对成本敏感的简单通信系统 |
| **DSB-SC (双边带抑制载波)** | $2\omega_m$ | • 无载波，功率效率高<br>• 比AM节省发射功率 | • 需要相干解调（载波同步）<br>• 接收端复杂<br>• 带宽仍较宽 | • 点对点通信<br>• 功率受限的应用 |
| **SSB (单边带调制)** | $\omega_m$ | • 带宽减半，频谱利用率最高<br>• 功率效率高 | • 滤波器实现困难<br>• 需要精确的载波同步<br>• 解调最复杂 | • 长距离通信（如短波通信）<br>• 频谱稀缺的场景<br>• 军用通信 |
| **FM (频率调制)** | 较宽（约 $2\Delta\omega$） | • 抗噪声能力强，音质好<br>• 捕获效应（强信号压制弱信号）<br>• 恒定包络，功率放大器效率高 | • 带宽宽，频谱利用率低<br>• 发射机复杂<br>• 邻道干扰 | • 广播电台（FM广播）<br>• 高保真音频传输<br>• 模拟电视伴音 |
| **PAM (脉冲幅度调制)** | 与采样率相关 | • 实现简单<br>• 易于时分复用（TDM）<br>• 与采样定理直接关联 | • 抗噪声性能差<br>• 产生码间串扰（ISI）<br>• 需要同步 | • 时分复用系统<br>• 数字通信的基础<br>• 模拟信号的数字化传输 |

**对比总结**：

| 维度 | AM | DSB-SC | SSB | FM | PAM |
|------|-----|--------|-----|-------|-----|
| **带宽效率** | 中 | 中 | 高（最优） | 低 | 中 |
| **功率效率** | 低 | 高 | 高 | 中 | 低 |
| **接收复杂度** | 低 | 高 | 最高 | 中 | 中 |
| **抗噪声性能** | 差 | 中 | 中 | 高 | 差 |
| **是否需载波同步** | 否（包络检波） | 是 | 是 | 否（鉴频器） | - |

## 综合练习题

### 题目1：信号基本变换

已知信号 $x(t)$ 的波形如下（假设 $x(t) = u(t) - u(t-1)$），画出以下信号的波形：
1. $x(t-1)$
2. $x(2t)$
3. $x(-t)$
4. $x(-2t+1)$

**答案**：
1. $x(t-1) = u(t-1) - u(t-2)$，波形向右平移1个单位
2. $x(2t) = u(2t) - u(2t-1) = u(t) - u(t-0.5)$，波形在水平方向压缩一半
3. $x(-t) = u(-t) - u(-t+1) = u(t) - u(t+1)$，波形关于纵轴翻转
4. $x(-2t+1) = u(-2t+1) - u(-2t) = u(t) - u(t-0.5)$，波形先翻转，再压缩2倍，再右移0.5单位

### 题目2：卷积计算

计算 $x(t) = e^{-t}u(t)$ 与 $h(t) = e^{-2t}u(t)$ 的卷积。

**答案**：
$$y(t) = x(t) * h(t) = \int_{-\infty}^{+\infty}e^{-\tau}u(\tau) \cdot e^{-2(t-\tau)}u(t-\tau)d\tau$$

当 $t < 0$ 时，无有效积分区间，$y(t) = 0$

当 $t \geq 0$ 时：
$$y(t) = \int_{0}^{t}e^{-\tau} \cdot e^{-2(t-\tau)}d\tau = e^{-2t}\int_{0}^{t}e^{\tau}d\tau$$
$$= e^{-2t}[e^{\tau}]_0^t = e^{-2t}(e^t - 1) = e^{-t} - e^{-2t}, \quad t \geq 0$$

---

### 题目3：离散卷积计算

计算 $x[n] = \{1, 2, 1\}$ 与 $h[n] = \{1, 1\}$ 的卷积。

**答案**：使用列表法
```
          h[0]  h[1]  h[2]  h[3]
            1     1     0     0
x[0] = 1    1     1
x[1] = 2          2     2
x[2] = 1                1     1
---------------------------------
           y[0]  y[1]  y[2]  y[3]
             1     3     3     1
```

$y[n] = \{1, 3, 3, 1\}$

### 题目4：系统性质判断

判断系统 $y[n] = x[n] - x[n-1]$ 的线性、时不变、因果、稳定性。

**答案**：
- 线性：$y[n] = \sum_{k=0}^{1}(-1)^k x[n-k]$，满足叠加性和齐次性，是线性系统
- 时不变：$y[n-n_0] = x[n-n_0] - x[n-n_0-1]$，而输入 $x[n-n_0]$ 产生的输出也是 $x[n-n_0] - x[n-n_0-1]$，是时不变系统
- 因果：输出只依赖当前输入 $x[n]$ 和过去输入 $x[n-1]$，是因果系统
- 稳定：若 $|x[n]| < M$，则 $|y[n]| = |x[n] - x[n-1]| \leq |x[n]| + |x[n-1]| < 2M$，是稳定系统

### 题目5：周期信号傅里叶级数

求周期信号 $x(t) = \begin{cases} 1, & |t| < 1 \\ 0, & 1 < |t| < 2 \end{cases}$（周期 $T=4$）的傅里叶系数 $a_k$。

**答案**：
基波频率 $\omega_0 = \frac{2\pi}{T} = \frac{\pi}{2}$

$$a_k = \frac{1}{T}\int_{-T/2}^{T/2}x(t)e^{-jk\omega_0 t}dt = \frac{1}{4}\int_{-1}^{1}1 \cdot e^{-jk\frac{\pi}{2}t}dt$$

$$= \frac{1}{4}\left[\frac{e^{-jk\frac{\pi}{2}t}}{-jk\frac{\pi}{2}}\right]_{-1}^{1} = \frac{1}{4}\cdot\frac{2}{jk\pi}(e^{-jk\frac{\pi}{2}} - e^{jk\frac{\pi}{2}})$$

$$= \frac{1}{jk\pi}\cdot\frac{e^{-jk\frac{\pi}{2}} - e^{jk\frac{\pi}{2}}}{2} = \frac{1}{jk\pi}\cdot(-j\sin(k\frac{\pi}{2}))$$

$$= \frac{\sin(k\frac{\pi}{2})}{k\pi}$$

当 $k = 0$ 时：
$$a_0 = \frac{1}{4}\int_{-1}^{1}1 dt = \frac{2}{4} = \frac{1}{2}$$

### 题目6：傅里叶变换计算

求 $x(t) = e^{-|t|}$ 的傅里叶变换 $X(j\omega)$。

**答案**：
$$X(j\omega) = \int_{-\infty}^{+\infty}e^{-|t|}e^{-j\omega t}dt = \int_{-\infty}^{0}e^{t}e^{-j\omega t}dt + \int_{0}^{+\infty}e^{-t}e^{-j\omega t}dt$$

$$= \int_{-\infty}^{0}e^{(1-j\omega)t}dt + \int_{0}^{+\infty}e^{-(1+j\omega)t}dt$$

$$= \left[\frac{e^{(1-j\omega)t}}{1-j\omega}\right]_{-\infty}^{0} + \left[\frac{e^{-(1+j\omega)t}}{-(1+j\omega)}\right]_{0}^{+\infty}$$

$$= \frac{1}{1-j\omega} - 0 + 0 - \frac{1}{-(1+j\omega)}$$

$$= \frac{1}{1-j\omega} + \frac{1}{1+j\omega} = \frac{1+j\omega + 1-j\omega}{1 + \omega^2}$$

$$= \frac{2}{1 + \omega^2}$$

### 题目7：傅里叶变换性质应用

已知信号 $x(t)$ 的傅里叶变换为 $X(j\omega)$，求 $y(t) = x(2t-1)$ 的傅里叶变换 $Y(j\omega)$。

**答案**：
利用时移和尺度变换性质：

先进行时移：$x(t-1) \leftrightarrow X(j\omega)e^{-j\omega}$

再进行尺度变换：$x(2t-1) \leftrightarrow \frac{1}{2}X(j\frac{\omega}{2})e^{-j\frac{\omega}{2}}$

因此：
$$Y(j\omega) = \frac{1}{2}X(j\frac{\omega}{2})e^{-j\frac{\omega}{2}}$$

### 题目8：采样定理应用

已知信号 $x(t) = \frac{\sin(100\pi t)}{100\pi t} \cdot \cos(2000\pi t)$，求无失真采样所需的最小采样频率。

**答案**：
信号包含两个分量：
1. $\frac{\sin(100\pi t)}{100\pi t}$：最高频率为 $100\pi$ rad/s
2. $\cos(2000\pi t)$：频率为 $2000\pi$ rad/s

由调制性质，$x(t)$ 的频谱是 $\frac{\sin(100\pi t)}{100\pi t}$ 的频谱平移到 $\pm 2000\pi$ 处。

信号最高频率：
$$\omega_m = 2000\pi + 100\pi = 2100\pi \text{ rad/s}$$
$$f_m = 1050 \text{ Hz}$$

奈奎斯特率：
$$2f_m = 2100 \text{ Hz}$$

最小采样频率应大于奈奎斯特率：
$$f_s > 2100 \text{ Hz}$$

### 题目9：AM调制

已知调制信号 $x(t) = \cos(100\pi t)$，载波 $c(t) = \cos(2000\pi t)$，进行DSB-SC调制，求已调信号的频谱。

**答案**：
已调信号：
$$y(t) = x(t)c(t) = \cos(100\pi t)\cos(2000\pi t)$$

利用三角恒等式：
$$y(t) = \frac{1}{2}[\cos(2100\pi t) + \cos(1900\pi t)]$$

频谱：
$$Y(j\omega) = \pi[\delta(\omega - 2100\pi) + \delta(\omega + 2100\pi)] + \pi[\delta(\omega - 1900\pi) + \delta(\omega + 1900\pi)]$$

信号带宽为 $200\pi$ rad/s（即100 Hz），中心频率为 $2000\pi$ rad/s（即1000 Hz）。

### 题目10：频域滤波

已知LTI系统的频率响应 $H(j\omega) = \begin{cases} 1, & |\omega| < 100 \\ 0, & |\omega| > 100 \end{cases}$，输入信号为 $x(t) = \frac{\sin(50t)}{50t}$，求输出 $y(t)$。

**答案**：
$x(t) = \frac{\sin(50t)}{50t}$ 的傅里叶变换为：
$$X(j\omega) = \begin{cases} \frac{\pi}{50}, & |\omega| < 50 \\ 0, & |\omega| > 50 \end{cases}$$

系统是截止频率为100 rad/s的理想低通滤波器，$x(t)$ 的频谱完全在通带内。

输出频谱：
$$Y(j\omega) = X(j\omega)H(j\omega) = X(j\omega)$$

因此输出等于输入：
$$y(t) = x(t) = \frac{\sin(50t)}{50t}$$

### 题目11：帕斯瓦尔定理

计算信号 $x(t) = e^{-t}u(t)$ 的能量。

**答案**：
**方法1：时域计算**
$$E = \int_{-\infty}^{+\infty}|x(t)|^2 dt = \int_{0}^{+\infty}e^{-2t}dt = \left[-\frac{1}{2}e^{-2t}\right]_0^{+\infty} = 0 - (-\frac{1}{2}) = \frac{1}{2}$$

**方法2：频域计算（帕斯瓦尔定理）**
$$X(j\omega) = \int_{0}^{+\infty}e^{-t}e^{-j\omega t}dt = \frac{1}{1+j\omega}$$
$$|X(j\omega)|^2 = \frac{1}{1 + \omega^2}$$

$$E = \frac{1}{2\pi}\int_{-\infty}^{+\infty}|X(j\omega)|^2 d\omega = \frac{1}{2\pi}\int_{-\infty}^{+\infty}\frac{1}{1+\omega^2}d\omega = \frac{1}{2\pi}[\arctan\omega]_{-\infty}^{+\infty}$$

$$= \frac{1}{2\pi}\left(\frac{\pi}{2} - (-\frac{\pi}{2})\right) = \frac{1}{2\pi} \cdot \pi = \frac{1}{2}$$

### 题目12：系统可逆性

已知LTI系统的冲激响应为 $h(t) = e^{-t}u(t)$，求逆系统的冲激响应。

**答案**：
设逆系统冲激响应为 $h_{inv}(t)$，则：
$$h(t) * h_{inv}(t) = \delta(t)$$

在频域：
$$H(j\omega)H_{inv}(j\omega) = 1$$

$$\frac{1}{1+j\omega}H_{inv}(j\omega) = 1$$

$$H_{inv}(j\omega) = 1 + j\omega$$

求逆傅里叶变换：
$$h_{inv}(t) = \delta(t) + \frac{d}{dt}\delta(t)$$

### 题目13：离散时间傅里叶变换

求 $x[n] = a^{|n|}$（$|a| < 1$）的离散时间傅里叶变换。

**答案**：
$$X(e^{j\omega}) = \sum_{n=-\infty}^{+\infty}a^{|n|}e^{-j\omega n} = \sum_{n=0}^{+\infty}a^{n}e^{-j\omega n} + \sum_{n=-\infty}^{-1}a^{-n}e^{-j\omega n}$$

令 $m = -n$，第二项：
$$\sum_{m=1}^{+\infty}a^{m}e^{j\omega m} = \sum_{m=1}^{+\infty}(ae^{j\omega})^m$$

$$X(e^{j\omega}) = \sum_{n=0}^{+\infty}(ae^{-j\omega})^n + \sum_{m=1}^{+\infty}(ae^{j\omega})^m$$

$$= \frac{1}{1 - ae^{-j\omega}} + \frac{ae^{j\omega}}{1 - ae^{j\omega}}$$

$$= \frac{1 - ae^{j\omega} + ae^{j\omega} - a^2}{(1 - ae^{-j\omega})(1 - ae^{j\omega})} = \frac{1 - a^2}{1 - 2a\cos\omega + a^2}$$

### 题目14：系统频率响应

已知LTI系统由微分方程 $\frac{dy(t)}{dt} + 2y(t) = x(t)$ 描述，求系统的频率响应和冲激响应。

**答案**：
对微分方程进行傅里叶变换：
$$j\omega Y(j\omega) + 2Y(j\omega) = X(j\omega)$$

频率响应：
$$H(j\omega) = \frac{Y(j\omega)}{X(j\omega)} = \frac{1}{j\omega + 2}$$

求逆傅里叶变换得到冲激响应：
$$h(t) = e^{-2t}u(t)$$

### 题目15：时分复用

某TDM系统复用4个信号，每个信号的带宽均为 4 kHz，采用奈奎斯特采样率，每个脉冲的宽度为 10 μs。求系统的最小带宽和可复用的最大信号数量。

**答案**：
1. 根据奈奎斯特采样定理，每个信号的采样率：
   $$f_s = 2 \times 4 = 8 \text{ kHz}$$

   采样周期：
   $$T = \frac{1}{f_s} = 125 \text{ μs}$$

2. 每个信号的时隙宽度：
   $$\frac{T}{4} = 31.25 \text{ μs}$$

   由于脉冲宽度 $\Delta = 10 \text{ μs} < 31.25 \text{ μs}$，满足要求。

3. 系统最小带宽：
   $$B \approx \frac{1}{\Delta} = \frac{1}{10 \times 10^{-6}} = 100 \text{ kHz}$$

4. 可复用的最大信号数量：
   $$N_{max} = \frac{T}{\Delta} = \frac{125}{10} = 12.5$$
   取整数，最多可复用 12 个信号。

## 附录：常用公式总结

### 信号分类速查
| 信号类型 | 定义 | 特点 |
|---------|------|------|
| 确定信号 | 对给定时间值确定 | $x(t)$ 无随机性 |
| 随机信号 | 对给定时间值不确定 | 如噪声 |
| 连续时间 | $x(t), t \in \mathbb{R}$ | 在连续时间定义 |
| 离散时间 | $x[n], n \in \mathbb{Z}$ | 在离散点定义 |
| 周期信号 | $x(t) = x(t+T)$ | $T$ 为周期 |
| 能量信号 | $E < \infty, P = 0$ | 能量有限 |
| 功率信号 | $P < \infty, E \to \infty$ | 功率有限 |

### 信号变换速查
| 变换 | 公式 | 效果 |
|------|------|------|
| 时移 | $x(t-t_0)$ | 右移 $t_0$ |
| 翻转 | $x(-t)$ | 关于纵轴翻转 |
| 尺度 | $x(at)$ | $a>1$ 压缩，$0<a<1$ 扩展 |

### 奇异信号速查
| 信号 | 定义 | 重要性质 |
|------|------|----------|
| $\delta(t)$ | $t=0$ 时 $\delta(0)=+\infty$，其他为0 | 抽样、偶对称 |
| $\delta[n]$ | $n=0$ 时 $\delta[0]=1$，其他为0 | 抽样 |
| $u(t)$ | $t<0$ 为0，$t>0$ 为1 | $u(t) = \int_{-\infty}^{t}\delta(\tau)d\tau$ |
| $u[n]$ | $n<0$ 为0，$n \geq 0$ 为1 | $u[n] = \sum_{k=-\infty}^{n}\delta[k]$ |

### 卷积公式速查
| 类型 | 公式 |
|------|------|
| 离散卷积 | $y[n] = \sum_{k=-\infty}^{+\infty}x[k]h[n-k]$ |
| 连续卷积 | $y(t) = \int_{-\infty}^{+\infty}x(\tau)h(t-\tau)d\tau$ |
| 与冲激卷积 | $x(t) * \delta(t-t_0) = x(t-t_0)$ |

### 系统性质判断速查
| 性质 | 判定条件 |
|------|----------|
| 线性 | $ax_1(t)+bx_2(t) \to ay_1(t)+by_2(t)$ |
| 无记忆 | 输出仅取决于当前输入 |
| 因果 | 输出只依赖过去和现在的输入 |
| 时不变 | $x(t-t_0) \to y(t-t_0)$ |
| 稳定 | 有界输入产生有界输出 |
| 可逆 | 输出可唯一确定输入 |

### 卷积与系统特性关系速查
| 特性 | 冲激响应条件 |
|------|--------------|
| 无记忆 | $h[n] = K\delta[n]$ 或 $h(t) = K\delta(t)$ |
| 因果 | $h[n] = 0 (n<0)$ 或 $h(t) = 0 (t<0)$ |
| 稳定 | $\sum_{n}|h[n]| < \infty$ 或 $\int_{-\infty}^{+\infty}|h(t)|dt < \infty$ |
| 可逆 | $h[n] * h_{inv}[n] = \delta[n]$ 或 $h(t) * h_{inv}(t) = \delta(t)$ |

### CTFS核心公式
- **傅里叶级数**：$x(t) = \sum_{k=-\infty}^{+\infty}a_k e^{jk\omega_0 t}$
- **傅里叶系数**：$a_k = \frac{1}{T}\int_{T}x(t)e^{-jk\omega_0 t}dt$
- **帕斯瓦尔定理**：$\frac{1}{T}\int_{T}|x(t)|^2 dt = \sum_{k=-\infty}^{+\infty}|a_k|^2$

### DTFS核心公式
- **傅里叶级数**：$x[n] = \sum_{k=<N>}a_k e^{jk\frac{2\pi}{N}n}$
- **傅里叶系数**：$a_k = \frac{1}{N}\sum_{n=<N>}x[n]e^{-jk\frac{2\pi}{N}n}$
- **帕斯瓦尔定理**：$\frac{1}{N}\sum_{n=<N>}|x[n]|^2 = \sum_{k=0}^{N-1}|a_k|^2$

### CTFT核心公式
- **傅里叶变换**：$X(j\omega) = \int_{-\infty}^{+\infty}x(t)e^{-j\omega t}dt$
- **傅里叶逆变换**：$x(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}X(j\omega)e^{j\omega t}d\omega$
- **帕斯瓦尔定理**：$\int_{-\infty}^{+\infty}|x(t)|^2 dt = \frac{1}{2\pi}\int_{-\infty}^{+\infty}|X(j\omega)|^2 d\omega$

### DTFT核心公式
- **傅里叶变换**：$X(e^{j\omega}) = \sum_{n=-\infty}^{+\infty}x[n]e^{-j\omega n}$
- **傅里叶逆变换**：$x[n] = \frac{1}{2\pi}\int_{\pi}^{\pi}X(e^{j\omega})e^{j\omega n}d\omega$
- **帕斯瓦尔定理**：$\sum_{n=-\infty}^{+\infty}|x[n]|^2 = \frac{1}{2\pi}\int_{\pi}^{\pi}|X(e^{j\omega})|^2 d\omega$

### 采样核心公式
- **理想采样**：$x_s(t) = x(t) \cdot \sum_{n=-\infty}^{+\infty}\delta(t - nT_s)$
- **采样信号频谱**：$X_s(j\omega) = \frac{1}{T_s}\sum_{k=-\infty}^{+\infty}X(j\omega - jk\omega_s)$
- **奈奎斯特采样定理**：$\omega_s > 2\omega_m$

### 重建核心公式
- **理想低通滤波器**：$H_r(j\omega) = \begin{cases} T_s, & |\omega| < \omega_c \\ 0, & \text{其他} \end{cases}$
- **重建时域**：$h_r(t) = \frac{\omega_c}{\pi}\cdot\frac{\sin(\omega_c t)}{\omega_c t}$

### AM调制核心公式
- **DSB-SC调制**：$y(t) = x(t)\cos(\omega_c t)$
- **DSB-SC频谱**：$Y(j\omega) = \frac{1}{2}X(j(\omega-\omega_c)) + \frac{1}{2}X(j(\omega+\omega_c))$
- **同步解调**：$y(t) \cdot \cos(\omega_c t) \to$ 低通滤波 $\to \frac{1}{2}x(t)$

### 常用变换对速查
| 时域 | 频域 |
|------|------|
| $\delta(t)$ | $1$ |
| $1$ | $2\pi\delta(\omega)$ |
| $e^{-at}u(t)$ | $\frac{1}{a+j\omega}$ |
| $u(t)$ | $\pi\delta(\omega) + \frac{1}{j\omega}$ |
| $\cos(\omega_0 t)$ | $\pi[\delta(\omega-\omega_0) + \delta(\omega+\omega_0)]$ |
| $\sin(\omega_0 t)$ | $\frac{\pi}{j}[\delta(\omega-\omega_0) - \delta(\omega+\omega_0)]$ |
| $\frac{\sin(at)}{at}$ | $\frac{\pi}{a}[u(\omega+a) - u(\omega-a)]$ |
| 周期冲激串 $\sum\delta(t-nT)$ | $\frac{2\pi}{T}\sum\delta(\omega-k\frac{2\pi}{T})$ |

### LTI系统频域分析
- **频率响应**：$H(j\omega) = \frac{Y(j\omega)}{X(j\omega)}$
- **时域卷积**：$y(t) = x(t) * h(t) \leftrightarrow Y(j\omega) = X(j\omega)H(j\omega)$
- **微分方程**：$\sum a_k\frac{d^ky}{dt^k} = \sum b_k\frac{d^kx}{dt^k} \leftrightarrow H(j\omega) = \frac{\sum b_k(j\omega)^k}{\sum a_k(j\omega)^k}$
- **差分方程**：$\sum a_k y[n-k] = \sum b_k x[n-k] \leftrightarrow H(e^{j\omega}) = \frac{\sum b_k e^{-j\omega k}}{\sum a_k e^{-j\omega k}}$

### TDM与FDM速查
| 特性 | TDM（时分复用） | FDM（频分复用） |
|------|----------------|----------------|
| 复用方式 | 时间分割 | 频率分割 |
| 需要同步 | 是（帧同步、位同步） | 否（仅相干解调时需载波同步） |
| 同步复杂性 | 高 | 低 |
| 失步后果 | 数据错乱 | 相干解调时信号失真 |
| 抗干扰能力 | 较好 | 受邻道干扰 |
| 带宽效率 | 高 | 中等 |
