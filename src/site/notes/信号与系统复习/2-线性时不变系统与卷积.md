## 二、线性时不变系统与卷积

### 1、卷积认识

#### （1）基本概念

一个 LTI 系统在时刻 $n$（或 $t$）的响应就是在时间上每一点的输入值所产生的各个响应在该时刻 $n$（或 $t$）的叠加。

**离散**：$y[n] = \sum_{k=-\infty}^{+\infty}x[k]h[n-k]$ 称为卷积和，记为 $y[n] = x[n] * h[n]$。

**连续**：$y(t) = \int_{-\infty}^{+\infty}x(\tau)h(t-\tau)d\tau$ 称为卷积积分，记为 $y(t) = x(t) * h(t)$。

任何信号 $x[n]$ / $x(t)$ 都可以被分解成移位加权的单位脉冲 / 冲激信号的线性组合；任意信号与单位冲激信号的卷积是其本身，即 $x[n] * \delta[n] = x[n]$ 或 $x(t) * \delta(t) = x(t)$。

#### （2）卷积公式证明

对于离散时间 LTI 系统，将输入信号 $x[n]$ 分解为移位加权的单位脉冲：
$$x[n] = \sum_{k=-\infty}^{+\infty}x[k]\delta[n-k]$$

根据线性和时不变性：
$$y[n] = \sum_{k=-\infty}^{+\infty}x[k] \cdot h[n-k] = x[n] * h[n]$$

对于连续时间 LTI 系统，同样分解：
$$x(t) = \int_{-\infty}^{+\infty}x(\tau)\delta(t-\tau)d\tau$$

$$y(t) = \int_{-\infty}^{+\infty}x(\tau)h(t-\tau)d\tau = x(t) * h(t)$$

#### （3）卷积基本性质

**交换律**

离散：$x_1[n] * x_2[n] = x_2[n] * x_1[n]$

连续：$x_1(t) * x_2(t) = x_2(t) * x_1(t)$

**分配律**

离散：$x_1[n] * (x_2[n] + x_3[n]) = x_1[n] * x_2[n] + x_1[n] * x_3[n]$

连续：$x_1(t) * (x_2(t) + x_3(t)) = x_1(t) * x_2(t) + x_1(t) * x_3(t)$

**结合律**

离散：$(x_1[n] * x_2[n]) * x_3[n] = x_1[n] * (x_2[n] * x_3[n])$

连续：$(x_1(t) * x_2(t)) * x_3(t) = x_1(t) * (x_2(t) * x_3(t))$

**延时卷积**：$x[n-n_0] * h[n] = y[n-n_0]$，离散同理。
$x(t-t_0) * h(t) = y(t-t_0)$

#### （4）卷积与系统基本特性

**无记忆性**：离散 $h[n] = K\delta[n]$，连续 $h(t) = K\delta(t)$，若 $h[n]/h(t)$ 在 $n \neq 0/t \neq 0$ 时不全为 0 则有记忆性。

**可逆性**：互逆系统级联输出等于原系统输入，即离散 $h[n] * h_{\text{逆}}[n] = \delta[n]$，连续 $h(t) * h_{\text{逆}}(t) = \delta(t)$

**因果性**：未发生输入之前不会响应，即离散 $h[n] = 0$（当 $n < 0$），连续 $h(t) = 0$（当 $t < 0$）。

**稳定性**：有界的输入产生有界的输出。
- 离散：$\sum_{n=-\infty}^{+\infty}|h[n]| < +\infty$
- 连续：$\int_{-\infty}^{+\infty}|h(t)|dt < +\infty$

#### 【练习题】

**题目20**：已知两个因果LTI系统冲激响应分别为 $h_1(t) = e^{-2t}u(t)$ 和 $h_2(t) = e^{-t}u(t)$，求级联系统的冲激响应。

**答案**：
级联系统的冲激响应为两个冲激响应的卷积：
$$h(t) = h_1(t) * h_2(t) = e^{-2t}u(t) * e^{-t}u(t)$$

利用公式 $e^{-at}u(t) * e^{-bt}u(t) = \frac{e^{-at} - e^{-bt}}{b - a}u(t)$（$a \neq b$）：
$$h(t) = \frac{e^{-2t} - e^{-t}}{1 - 2}u(t) = (e^{-t} - e^{-2t})u(t)$$

---

**题目21**：判断系统冲激响应 $h[n] = 2^n u[n]$ 的因果性和稳定性。

**答案**：
- 因果性：$h[n] = 0$ 当 $n < 0$，系统是因果的
- 稳定性：
  $$\sum_{n=-\infty}^{+\infty}|h[n]| = \sum_{n=0}^{+\infty}2^n = +\infty$$
  系统不稳定

---

**题目22**：判断系统冲激响应 $h(t) = \frac{\sin(2t)}{2t}$ 的稳定性。

**答案**：
$$\int_{-\infty}^{+\infty}|h(t)|dt = \int_{-\infty}^{+\infty}\left|\frac{\sin(2t)}{2t}\right|dt$$

由于 $\int_{-\infty}^{+\infty}\left|\frac{\sin t}{t}\right|dt = +\infty$，系统不稳定。

### 2、卷积计算

#### （1）离散信号列表法（浙大胡浩基老师视频介绍）

#### 【练习题】

**题目23**：计算 $x[n] = \{1, 2, 1, 0, -1\}$ 与 $h[n] = \{1, 1\}$ 的卷积（$x[0] = 1$, $x[1] = 2$ 等）

**答案**：

```
         h[-1]  h[0]  h[1]  h[2]  h[3]  h[4]
           1     1     0     0     0     0
x[0] = 1   1     1
x[1] = 2         2     2
x[2] = 1               1     1
x[3] = 0                     0     0
x[4] = -1                          -1    -1
------------------------------------------------
         y[0]   y[1]  y[2]  y[3]  y[4]  y[5]
           1     3     3     1    -1    -1
```

因此这两个序列卷积和结果为：$y[n] = \{1, 3, 3, 1, -1, -1\}$

#### （2）连续信号图解法

**步骤**：
1. **换元**：$x(t)$ 中 $t$ 换成 $\tau$，$h(t)$ 中 $t$ 换成 $\tau$
2. **反折**：将 $h(\tau)$ 翻转得到 $h(-\tau)$
3. **平移**：将 $h(-\tau)$ 平移 $t$ 得到 $h(t-\tau)$
4. **相乘**：求 $x(\tau)h(t-\tau)$
5. **积分**：求 $\int_{-\infty}^{+\infty}x(\tau)h(t-\tau)d\tau$

#### 【练习题】

**题目24**：计算 $x(t) = u(t) - u(t-1)$ 与 $h(t) = u(t) - u(t-2)$ 的卷积。

**答案**：
1. $x(\tau) = u(\tau) - u(\tau-1)$，$h(\tau) = u(\tau) - u(\tau-2)$
2. $h(-\tau) = u(-\tau) - u(-\tau-2)$
3. $h(t-\tau) = u(t-\tau) - u(t-\tau-2)$

分段分析：
- $t < 0$：无重叠，$y(t) = 0$
- $0 \leq t < 1$：$x(\tau)h(t-\tau) = 1$，积分限 $[0, t]$，$y(t) = t$
- $1 \leq t < 2$：$x(\tau)h(t-\tau) = 1$，积分限 $[0, 1]$，$y(t) = 1$
- $2 \leq t < 3$：$x(\tau)h(t-\tau) = 1$，积分限 $[t-2, 1]$，$y(t) = 3-t$
- $t \geq 3$：无重叠，$y(t) = 0$

得到的卷积和信号的长度等于参与卷积的信号的长度和减去 1（此处说的信号长度是从第一个非零点到最后一个非零点的长度，中间的零值算在内）。

$$y(t) = \begin{cases} t, & 0 \leq t < 1 \\ 1, & 1 \leq t < 2 \\ 3-t, & 2 \leq t < 3 \\ 0, & \text{其他} \end{cases}$$

#### （3）公式法

**离散**：计算 $y[n] = x[n] * h[n]$

先将两个序列的括号内用 $k$ 和 $n-k$ 代替，以 $n$ 作为自变量作图分析，后者相对于前者进行了反折并平移 $n$ 个单位，接着按公式 $y[n] = \sum_{k}x[k]h[n-k]$ 求各和。

#### 【练习题】

**题目25**：计算 $x[n] = \alpha^n u[n]$ 与 $h[n] = \beta^n u[n]$ 的卷积（假设 $\alpha \neq \beta$）。

**答案**：
$$y[n] = \sum_{k=-\infty}^{+\infty}\alpha^k u[k] \cdot \beta^{n-k}u[n-k]$$

由于 $u[k]$ 要求 $k \geq 0$，$u[n-k]$ 要求 $n-k \geq 0$ 即 $k \leq n$。

当 $n < 0$ 时，无有效 $k$，$y[n] = 0$

当 $n \geq 0$ 时：
$$y[n] = \sum_{k=0}^{n}\alpha^k \beta^{n-k} = \beta^n\sum_{k=0}^{n}\left(\frac{\alpha}{\beta}\right)^k$$

$$= \beta^n \cdot \frac{1 - (\alpha/\beta)^{n+1}}{1 - \alpha/\beta} = \frac{\beta^{n+1} - \alpha^{n+1}}{\beta - \alpha}, \quad n \geq 0$$

---

**连续**：计算 $y(t) = x(t) * h(t)$

先将两个信号的括号内用 $\tau$ 和 $t-\tau$ 代替，以 $t$ 作为自变量作图分析，后者相对于前者进行了反折并平移 $t$ 个单位，按公式 $y(t) = \int_{-\infty}^{+\infty}x(\tau)h(t-\tau)d\tau$ 求积分。

#### 【练习题】

**题目26**：计算 $x(t) = e^{-at}u(t)$ 与 $h(t) = e^{-bt}u(t)$ 的卷积（假设 $a \neq b$）。

**答案**：
$$y(t) = \int_{-\infty}^{+\infty}e^{-a\tau}u(\tau) \cdot e^{-b(t-\tau)}u(t-\tau)d\tau$$

由于 $u(\tau)$ 要求 $\tau \geq 0$，$u(t-\tau)$ 要求 $t-\tau \geq 0$ 即 $\tau \leq t$。

当 $t < 0$ 时，无有效 $\tau$，$y(t) = 0$

当 $t \geq 0$ 时：
$$y(t) = \int_{0}^{t}e^{-a\tau} \cdot e^{-b(t-\tau)}d\tau = e^{-bt}\int_{0}^{t}e^{-(a-b)\tau}d\tau$$

若 $a \neq b$：
$$y(t) = e^{-bt} \cdot \left[\frac{e^{-(a-b)\tau}}{-(a-b)}\right]_0^t = e^{-bt} \cdot \frac{1 - e^{-(a-b)t}}{a-b}$$

$$= \frac{e^{-bt} - e^{-at}}{a - b} = \frac{e^{-at} - e^{-bt}}{b - a}, \quad t \geq 0$$

若 $a = b$：
$$y(t) = e^{-at}\int_{0}^{t}d\tau = te^{-at}, \quad t \geq 0$$
