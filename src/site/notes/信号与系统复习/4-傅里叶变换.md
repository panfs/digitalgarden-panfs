## 四、非周期信号的傅里叶变换

### 1、连续时间傅里叶变换（CTFT）

#### （1）傅里叶变换认识

一个周期信号的周期 $T \to +\infty$，则周期信号将演变成一个非周期信号。

把非周期信号看成是周期信号在周期趋于无穷大时的极限，从而考查连续时间傅里叶级数在 $T \to +\infty$ 时的变化，就应该能够得到对非周期信号的频域表示方法。

周期信号的频谱就是与它相对应的非周期信号频谱的抽样。而非周期信号的频谱是对应的周期信号频谱的包络。

#### （2）连续时间傅里叶变换对

定义 $Ta_k$ 的包络函数/频谱密度函数：

$$X(j\omega) = \int_{-\infty}^{+\infty}x(t)e^{-j\omega t}dt$$

**傅里叶变换**：$X(j\omega) = \int_{-\infty}^{+\infty}x(t)e^{-j\omega t}dt$ 即时域转频域分析

**傅里叶逆变换**：$x(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}X(j\omega)e^{j\omega t}d\omega$ 即频域转时域分析

这一对关系被称为**连续时间傅里叶变换对**，简记为：
$$x(t) \leftrightarrow X(j\omega)$$

#### （3）$X(j\omega)$ 认识

一般而言 $X(j\omega)$ 是个复变函数：
$$X(j\omega) = |X(j\omega)|e^{j\varphi(j\omega)}$$

- $|X(j\omega)|$ 是**幅度谱**，描述了信号的幅频特性
- $\varphi(j\omega)$ 是**相位谱**，描述了信号的相频特性

#### （4）傅里叶变换存在的条件

傅里叶变换存在的充分条件是信号绝对可积：
$$\int_{-\infty}^{+\infty}|x(t)|dt < +\infty$$

#### 【练习题】

**题目49**：计算 $x(t) = \begin{cases} 1, & |t| < 1 \\ 0, & \text{其他} \end{cases}$ 的傅里叶变换。

**答案**：
$$X(j\omega) = \int_{-1}^{1}e^{-j\omega t}dt = \frac{2\sin\omega}{\omega} = 2\cdot\frac{\sin\omega}{\omega}$$

---

**题目50**：已知 $x(t) = e^{-at}u(t)$（$a>0$），求其傅里叶变换。

**答案**：
$$X(j\omega) = \int_{0}^{+\infty}e^{-at}e^{-j\omega t}dt = \frac{1}{a + j\omega}$$

---

**题目51**：求周期冲激串 $y(t)=\sum_{n=-\infty}^{\infty}\delta\left(t-n T-\frac{T}{2}\right)$ 的傅里叶变换。

**答案**：
$$Y(j\omega) = \frac{2\pi}{T}\sum_{k=-\infty}^{+\infty}(-1)^k\delta\left(\omega - k\frac{2\pi}{T}\right)$$

### 2、离散时间傅里叶变换（DTFT）

#### （1）DTFT定义

离散时间傅里叶变换定义为：

$$X(e^{j\omega}) = \sum_{n=-\infty}^{+\infty}x[n]e^{-j\omega n}$$

其中 $\omega$ 是数字频率，单位为弧度。

**傅里叶逆变换（IDTFT）**：
$$x[n] = \frac{1}{2\pi}\int_{-\pi}^{\pi}X(e^{j\omega})e^{j\omega n}d\omega$$

记为：
$$x[n] \leftrightarrow X(e^{j\omega})$$

#### （2）DTFT的特点

- **周期性**：$X(e^{j\omega})$ 是以 $2\pi$ 为周期的函数
$$X(e^{j(\omega + 2\pi k)}) = X(e^{j\omega}), \quad k \in \mathbb{Z}$$

- **共轭对称性**：对于实信号 $x[n]$：
$$X(e^{-j\omega}) = X^*(e^{j\omega})$$

即：实部是偶函数，虚部是奇函数。

#### （3）DTFT与CTFT的关系

若 $x[n]$ 是连续时间信号 $x(t)$ 的采样，采样周期为 $T_s$，则其DTFT与CTFT存在关系。详细内容请参考《采样与通信系统》笔记。

#### 【练习题】

**题目52**：计算 $x[n] = a^n u[n]$（$|a|<1$）的DTFT。

**答案**：
$$X(e^{j\omega}) = \sum_{n=0}^{+\infty}a^n e^{-j\omega n} = \frac{1}{1-ae^{-j\omega}}$$

---

**题目53**：已知 $x[n] = \delta[n]$，求其DTFT。

**答案**：
$$X(e^{j\omega}) = 1$$

---

**题目54**：计算 $x[n] = u[n]$ 的DTFT。

**答案**：
$$X(e^{j\omega}) = \frac{1}{1-e^{-j\omega}} + \pi\sum_{k=-\infty}^{+\infty}\delta(\omega - 2\pi k)$$

#### （4）离散时间滤波器类型快速判断（考试实用）

**核心思想**：通过测试两个特殊频率（DC 和 $\omega=\pi$）的响应，快速判断离散时间滤波器的类型。

**测试信号**：

1. **DC（直流）**：$x[n] = 1$
   - 频率为 $\omega = 0$
   - 对应数字频率的最低频率

2. **最高频**：$x[n] = (-1)^n$
   - 频率为 $\omega = \pi$
   - 对应数字频率的最高频率

**判断规则**：

| DC响应 | $\omega=\pi$响应 | 滤波器类型 |
|--------|-----------------|-----------|
| 通过 | 不通过 | Low-pass（低通） |
| 不通过 | 通过 | High-pass（高通） |
| 通过 | 通过 | Band-pass（带通） |
| 不通过 | 不通过 | Band-stop（带阻） |

**差分结构快速识别表**：

| 差分结构 | 滤波器类型 | 说明 |
|---------|-----------|------|
| $x[n] + x[n-1]$ | Low-pass | 对称求和，平滑作用 |
| $x[n] - x[n-1]$ | High-pass | 一阶差分，提取变化 |
| $x[n] - 2x[n-1] + x[n-2]$ | High-pass | 二阶差分 |
| 对称平均（如 $\frac{x[n] + x[n-1]}{2}$） | Low-pass | 移动平均滤波器 |
| 正负交替系数 | High-pass | 高频分量通过 |
| 中间减两边 | Band-pass | 中频通过 |

**示例应用**：

**示例1**：判断 $y[n] = x[n] + x[n-1]$ 的滤波器类型

- 测试DC：$x[n] = 1$ → $y[n] = 1 + 1 = 2$（通过）
- 测试$\omega=\pi$：$x[n] = (-1)^n$ → $y[n] = (-1)^n + (-1)^{n-1} = (-1)^n - (-1)^n = 0$（不通过）
- 结论：**Low-pass（低通）**

**示例2**：判断 $y[n] = x[n] - x[n-1]$ 的滤波器类型

- 测试DC：$x[n] = 1$ → $y[n] = 1 - 1 = 0$（不通过）
- 测试$\omega=\pi$：$x[n] = (-1)^n$ → $y[n] = (-1)^n - (-1)^{n-1} = (-1)^n - (-1)^n \cdot (-1)^{-1} = (-1)^n - (-1)^n \cdot (-1) = (-1)^n + (-1)^n = 2(-1)^n$（通过）
- 结论：**High-pass（高通）**

**方法优势**：
- 快速：无需计算完整的频率响应 $H(e^{j\omega})$
- 直观：直接理解滤波器的频率选择特性
- 适用：适用于差分方程描述的离散时间LTI系统

**注意**：
- 该方法适用于**线性时不变（LTI）离散系统**
- 测试结果基于系统在$\omega=0$和$\omega=\pi$处的幅度响应
- "通过"指输出信号不为零（或增益较大），"不通过"指输出为零（或增益很小）

### 3、典型信号的傅里叶变换对

#### （1）CTFT基本变换对

| 时域信号 $x(t)$ | 频域变换 $X(j\omega)$ | 收敛条件 |
|---------------|----------------|-----------|
| $\delta(t)$ | $1$ | - |
| $1$ | $2\pi\delta(\omega)$ | - |
| $e^{-at}u(t)$（$a>0$）| $\frac{1}{a+j\omega}$ | - |
| $e^{-at}u(-t)$（$a>0$）| $\frac{1}{-a+j\omega}$ | - |
| $e^{-a|t|}$（$a>0$）| $\frac{2a}{a^2+\omega^2}$ | 双边指数衰减 |
| $u(t)$ | $\pi\delta(\omega) + \frac{1}{j\omega}$ | - |
| $\text{sgn}(t)$ | $\frac{2}{j\omega}$ | - |
| $\frac{1}{t}$ | $-j\pi\text{sgn}(\omega)$ | 由对偶性从 $\text{sgn}(t)$ 导出 |
| $\cos(\omega_0 t)$ | $\pi[\delta(\omega-\omega_0) + \delta(\omega+\omega_0)]$ | - |
| $\sin(\omega_0 t)$ | $\frac{\pi}{j}[\delta(\omega-\omega_0) - \delta(\omega+\omega_0)]$ | - |

#### （2）DTFT基本变换对

| 时域信号 $x[n]$ | 频域变换 $X(e^{j\omega})$ |
|---------------|---------------------|
| $\delta[n]$ | $1$ |
| $1$ | $2\pi\sum_{k=-\infty}^{+\infty}\delta(\omega-2\pi k)$ |
| $a^n u[n]$（$|a|<1$）| $\frac{1}{1-ae^{-j\omega}}$ |
| $u[n]$ | $\frac{1}{1-e^{-j\omega}} + \pi\sum_{k\neq 0}\delta(\omega-2\pi k)$ |
| $\cos(\omega_0 n)$ | $\pi\sum_{k=-\infty}^{+\infty}[\delta(\omega-\omega_0-2\pi k) + \delta(\omega+\omega_0-2\pi k)]$ |
| $\sin(\omega_0 n)$ | $\frac{\pi}{j}\sum_{k=-\infty}^{+\infty}[\delta(\omega-\omega_0-2\pi k) - \delta(\omega+\omega_0-2\pi k)]$ |

#### （3）重要公式

**矩形脉冲**（宽度为 $\tau$）：
$$x(t) = \begin{cases} 1, & |t| < \frac{\tau}{2} \\ 0, & \text{其他} \end{cases} \leftrightarrow X(j\omega) = \tau\cdot\frac{\sin(\omega\tau/2)}{\omega\tau/2} = \frac{2\sin(\omega\tau/2)}{\omega}$$

**周期冲激串**：
$$\sum_{n=-\infty}^{+\infty}\delta(t - nT) \leftrightarrow \frac{2\pi}{T}\sum_{k=-\infty}^{+\infty}\delta(\omega - k\omega_0)$$

其中 $\omega_0 = \frac{2\pi}{T}$。

#### 【练习题】

**题目55**：求 $x(t) = \sin\left(\frac{\pi}{2} t\right)+\cos(t)+\delta(t)$ 的傅里叶变换。

**答案**：
$$X(j\omega) = \frac{\pi}{j}[\delta(\omega-\frac{\pi}{2}) - \delta(\omega+\frac{\pi}{2})] + \pi[\delta(\omega-1) + \delta(\omega+1)] + 1$$

---

**题目56**：求 $\mathcal{F}\{\frac{\sin t}{t}\}$。

**答案**：
$$\mathcal{F}\{\frac{\sin t}{t}\} = \pi[u(\omega+1) - u(\omega-1)]$$

### 4、傅里叶变换的性质

#### （1）CTFT性质汇总

| 性质 | 时域 | 频域 |
|------|------|------|
| 线性 | $ax_1(t) + bx_2(t)$ | $aX_1(j\omega) + bX_2(j\omega)$ |
| 时移 | $x(t-t_0)$ | $X(j\omega)e^{-j\omega t_0}$ |
| 频移 | $x(t)e^{j\omega_0 t}$ | $X(j(\omega-\omega_0))$ |
| 尺度变换 | $x(at)$ | $\frac{1}{|a|}X\left(\frac{j\omega}{a}\right)$ |
| 时域微分 | $\frac{d}{dt}x(t)$ | $j\omega X(j\omega)$ |
| 时域积分 | $\int_{-\infty}^{t}x(\tau)d\tau$ | $\frac{X(j\omega)}{j\omega} + \pi X(0)\delta(\omega)$ |
| 对偶性 | 若 $x(t) \leftrightarrow X(j\omega)$ | 则 $X(jt) \leftrightarrow 2\pi x(-\omega)$ |
| 卷积 | $x_1(t) * x_2(t)$ | $X_1(j\omega)X_2(j\omega)$ |
| 乘积 | $x_1(t) \cdot x_2(t)$ | $\frac{1}{2\pi}X_1(j\omega) * X_2(j\omega)$ |
| 帕斯瓦尔定理 | $\int_{-\infty}^{+\infty}|x(t)|^2 dt$ | $\frac{1}{2\pi}\int_{-\infty}^{+\infty}|X(j\omega)|^2 d\omega$ |

#### （2）DTFT性质汇总

| 性质 | 时域 | 频域 |
|------|------|------|
| 线性 | $ax_1[n] + bx_2[n]$ | $aX_1(e^{j\omega}) + bX_2(e^{j\omega})$ |
| 时移 | $x[n-n_0]$ | $X(e^{j\omega})e^{-j\omega n_0}$ |
| 频移（调制）| $x[n]e^{j\omega_0 n}$ | $X(e^{j(\omega-\omega_0)})$ |
| 时域微分 | $\nabla x[n]$（离散微分）| $(1-e^{-j\omega})X(e^{j\omega})$ |
| 时域卷积 | $x_1[n] \circledast x_2[n]$ | $X_1(e^{j\omega})X_2(e^{j\omega})$ |
| 帕斯瓦尔定理 | $\sum_{n=-\infty}^{+\infty}|x[n]|^2$ | $\frac{1}{2\pi}\int_{-\pi}^{\pi}|X(e^{j\omega})|^2 d\omega$ |

#### （3）共轭对称性认识

$x^*(t)$ 是 $x(t)$ 的共轭信号，共轭对称性主要讨论当时域信号为奇信号、偶信号、实信号时，其频谱所具有的对称性。

由 $X(j\omega) = \int_{-\infty}^{+\infty}x(t)e^{-j\omega t}dt$ 可得：
$$X^*(j\omega) = \int_{-\infty}^{+\infty}x^*(t)e^{j\omega t}dt$$

所以 $X^*(-j\omega) = \int_{-\infty}^{+\infty}x^*(t)e^{-j\omega t}dt$，即得证：
$$x^*(t) \leftrightarrow X^*(-j\omega)$$

若 $x(t)$ 是实信号，则 $x(t) = x^*(t)$，有：
$$X(j\omega) = X^*(-j\omega)$$

即：
- 实部 $\text{Re}[X(j\omega)]$ 是偶函数
- 虚部 $\text{Im}[X(j\omega)]$ 是奇函数

若 $X(j\omega) = |X(j\omega)|e^{j\angle X(j\omega)}$，则：
- 模 $|X(j\omega)|$ 是偶函数
- 相位 $\angle X(j\omega)$ 是奇函数

**重要结论**：
- 实偶信号的傅里叶变换是实偶函数
- 实奇信号的傅里叶变换是纯虚奇函数

#### （4）尺度变换性质认识

时间坐标的尺度变化会改变信号变换的快慢，当时间坐标尺度压缩时，信号变化加快，频率提高，当时间坐标尺度扩展时，信号变化减慢，频率也就降低了。

信号如果在时域扩展 $a$ 倍，则其带宽相应压缩 $a$ 倍，反之亦然。

$$x(at) \leftrightarrow \frac{1}{|a|}X\left(\frac{j\omega}{a}\right)$$

**时频对偶性**：
- 时域压缩 $\leftrightarrow$ 频域扩展
- 时域扩展 $\leftrightarrow$ 频域压缩

#### （5）对偶性

若 $x(t) \leftrightarrow X(j\omega)$，则：
$$X(jt) \leftrightarrow 2\pi x(-\omega)$$

这体现了时域和频域的对偶性。

#### （6）帕斯瓦尔定理（Parseval's Theorem）

若 $x(t) \leftrightarrow X(j\omega)$，则：
$$\int_{-\infty}^{+\infty}|x(t)|^2 dt = \frac{1}{2\pi}\int_{-\infty}^{+\infty}|X(j\omega)|^2 d\omega$$

这表明信号的能量既可以在时域求得，也可以在频域求得。

由于 $|X(j\omega)|^2$ 表示了信号能量在频域的分布，因而称其为**能量谱密度**函数。

**离散时间版本**：
$$\sum_{n=-\infty}^{+\infty}|x[n]|^2 = \frac{1}{2\pi}\int_{-\pi}^{\pi}|X(e^{j\omega})|^2 d\omega$$

#### 【练习题】

**题目57**：利用频移性质求 $\mathcal{F}\{x(t)\cos(10t)\}$，已知 $\mathcal{F}\{x(t)\} = X(j\omega)$。

**答案**：
$$\mathcal{F}\{x(t)\cos(10t)\} = \frac{1}{2}X(j(\omega-10)) + \frac{1}{2}X(j(\omega+10))$$

### 5、频域卷积特性分析

#### （1）时域卷积 $\leftrightarrow$ 频域乘积

若 $x(t) \leftrightarrow X(j\omega)$，$h(t) \leftrightarrow H(j\omega)$

则有：
$$x(t) * h(t) \leftrightarrow X(j\omega)H(j\omega)$$

**离散时间**：
$$x_1[n] * x_2[n] \leftrightarrow X_1(e^{j\omega})X_2(e^{j\omega})$$

本质上卷积特性的成立正是因为复指数信号是一切 LTI 系统的特征函数。

#### （2）LTI系统的频域分析法

1. 由 $x(t) \leftrightarrow X(j\omega)$，根据系统的描述求出 $H(j\omega)$
2. 根据 $Y(j\omega) = X(j\omega)H(j\omega)$ 再逆变换求 $y(t) = \mathcal{F}^{-1}[Y(j\omega)]$

系统的频率响应：
$$H(j\omega) = \frac{Y(j\omega)}{X(j\omega)}$$

#### （3）时域乘积 $\leftrightarrow$ 频域卷积

若 $x_1(t) \leftrightarrow X_1(j\omega)$，$x_2(t) \leftrightarrow X_2(j\omega)$

则：
$$x_1(t) \cdot x_2(t) \leftrightarrow \frac{1}{2\pi}X_1(j\omega) * X_2(j\omega)$$

即：时域的乘积对应频域的卷积（除以 $2\pi$）。

#### （4）LTI系统对余弦信号的响应

**问题类型**：给定由多个余弦分量组成的输入信号，通过理想低通滤波器后的输出结果。

**一般形式**：
输入信号：$w(t) = \sum_k a_k \cos(\omega_k t)$

通过滤波器：$H(j\omega) = \begin{cases} K, & |\omega| < \omega_c \\ 0, & \text{其他} \end{cases}$

**分析方法**：

1. **识别每个余弦分量的频率**：$\cos(\omega_k t)$ 的频率为 $\omega_k$

2. **应用滤波器的频域特性**：
   - **通过**：$|\omega| < \omega_c$ 的频率分量，幅度乘以增益 $K$
   - **阻止**：$|\omega| \geq \omega_c$ 的频率分量，完全滤除

**输出结果**：
$$y(t) = \sum_{|\omega_k| < \omega_c} K \cdot a_k \cos(\omega_k t)$$

即：
- **保留**：所有频率满足 $|\omega_k| < \omega_c$ 的余弦分量，幅度乘以增益 $K$
- **滤除**：所有频率满足 $|\omega_k| \geq \omega_c$ 的余弦分量，输出为0

**物理含义**：
理想低通滤波器相当于一个**频率选择器**——只允许特定频率范围内的信号通过，其余全部静音。

**注意**：余弦信号 $\cos(\omega_k t)$ 的频谱包含两个冲激：$+\omega_k$ 和 $-\omega_k$，因此需要判断 $| \pm \omega_k | < \omega_c$，即 $|\omega_k| < \omega_c$。

#### 【练习题】

**题目61**：已知 $x(t) = e^{-t}u(t)$，$y(t) = x(t) \cdot \cos(5t)$，利用时域乘积性质求 $Y(j\omega)$。

**答案**：
利用 $x_1(t) \cdot x_2(t) \leftrightarrow \frac{1}{2\pi}X_1(j\omega) * X_2(j\omega)$

已知：
$$X(j\omega) = \mathcal{F}\{e^{-t}u(t)\} = \frac{1}{1+j\omega}$$
$$\mathcal{F}\{\cos(5t)\} = \pi[\delta(\omega-5) + \delta(\omega+5)]$$

应用乘积性质：
$$Y(j\omega) = \frac{1}{2\pi} \cdot X(j\omega) * \mathcal{F}\{\cos(5t)\}$$
$$= \frac{1}{2\pi} \cdot \frac{1}{1+j\omega} * \pi[\delta(\omega-5) + \delta(\omega+5)]$$
$$= \frac{1}{2} \left[\frac{1}{1+j(\omega-5)} + \frac{1}{1+j(\omega+5)}\right]$$

**题目62**：已知 $x(t) = e^{-t}u(t)$，$h(t) = e^{-2t}u(t)$，利用频域求 $y(t) = x(t) * h(t)$。

**答案**：
$$X(j\omega) = \frac{1}{1+j\omega}, \quad H(j\omega) = \frac{1}{2+j\omega}$$
$$Y(j\omega) = \frac{1}{(1+j\omega)(2+j\omega)} = \frac{1}{1+j\omega} - \frac{1}{2+j\omega}$$
$$y(t) = e^{-t}u(t) - e^{-2t}u(t)$$

**题目63**：已知离散时间LTI系统的冲激响应为 $h[n] = 0.2^n u[n]$，频率响应为 $H(e^{j\omega}) = \frac{1}{1-0.2e^{-j\omega}}$，求系统对输入 $x[n] = \cos(\omega_0 n)$ 的输出。

**答案**：
对于正弦输入信号，系统输出为：
$$y[n] = |H(e^{j\omega_0})| \cos(\omega_0 n + \angle H(e^{j\omega_0}))$$

计算系统的幅度响应：
$$|H(e^{j\omega_0})| = \left|\frac{1}{1-0.2e^{-j\omega_0}}\right| = \frac{1}{|1-0.2\cos\omega_0 + j0.2\sin\omega_0|}$$
$$= \frac{1}{\sqrt{(1-0.2\cos\omega_0)^2 + (0.2\sin\omega_0)^2}} = \frac{1}{\sqrt{1 - 0.4\cos\omega_0 + 0.04}}$$

计算系统的相位响应：
$$\angle H(e^{j\omega_0}) = -\arg(1-0.2e^{-j\omega_0}) = -\arctan\left(\frac{0.2\sin\omega_0}{1-0.2\cos\omega_0}\right)$$

因此：
$$y[n] = \frac{1}{\sqrt{1 - 0.4\cos\omega_0 + 0.04}} \cos\left(\omega_0 n - \arctan\frac{0.2\sin\omega_0}{1-0.2\cos\omega_0}\right)$$

### 6、傅里叶变换与傅里叶级数的关系

#### （1）傅里叶变换是傅里叶级数的极限形式

当周期 $T \to \infty$ 时，$\omega_0 = \frac{2\pi}{T} \to 0$，离散频率 $k\omega_0$ 变为连续频率 $\omega$。

傅里叶系数 $a_k$ 与傅里叶变换 $X(j\omega)$ 的关系：
$$a_k = \frac{1}{T}X(j\omega)\big|_{\omega=k\omega_0} = \frac{\omega_0}{2\pi}X(jk\omega_0)$$

#### （2）周期信号的傅里叶变换

对于周期信号 $x(t)$，其傅里叶变换由冲激函数组成：
$$X(j\omega) = 2\pi\sum_{k=-\infty}^{+\infty}a_k\delta(\omega - k\omega_0)$$

其中 $a_k$ 是傅里叶系数。

**例子**：$\cos(\omega_0 t)$ 的傅里叶变换
$$\mathcal{F}\{\cos(\omega_0 t)\} = \pi[\delta(\omega-\omega_0) + \delta(\omega+\omega_0)]$$

#### （3）非周期信号的频谱密度

非周期信号的傅里叶变换 $X(j\omega)$ 表示**频谱密度**，单位为[信号单位/频率单位]。

而周期信号的傅里叶系数 $a_k$ 表示**频谱强度**，单位为[信号单位]。

#### 【练习题】

**题目64**：已知周期信号 $x(t)$ 的傅里叶系数为 $a_k = \begin{cases} 1, & |k| = 1 \\ \frac{1}{2}, & |k| = 2 \\ 0, & \text{其他} \end{cases}$，求其傅里叶变换 $X(j\omega)$。

**答案**：
$$X(j\omega) = 2\pi\left[\delta(\omega-\omega_0) + \delta(\omega+\omega_0) + \frac{1}{2}\delta(\omega-2\omega_0) + \frac{1}{2}\delta(\omega+2\omega_0)\right]$$

**题目65**：已知周期冲激串 $x(t) = \sum_{n=-\infty}^{+\infty}\delta(t - nT)$，求其傅里叶变换。

**答案**：
$$X(j\omega) = \frac{2\pi}{T}\sum_{k=-\infty}^{+\infty}\delta\left(\omega - k\frac{2\pi}{T}\right)$$

## 附录：常用公式总结

### CTFT核心公式
- **傅里叶变换**：$X(j\omega) = \int_{-\infty}^{+\infty}x(t)e^{-j\omega t}dt$
- **傅里叶逆变换**：$x(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}X(j\omega)e^{j\omega t}d\omega$
- **帕斯瓦尔定理**：$\int_{-\infty}^{+\infty}|x(t)|^2 dt = \frac{1}{2\pi}\int_{-\infty}^{+\infty}|X(j\omega)|^2 d\omega$

### DTFT核心公式
- **傅里叶变换**：$X(e^{j\omega}) = \sum_{n=-\infty}^{+\infty}x[n]e^{-j\omega n}$
- **傅里叶逆变换**：$x[n] = \frac{1}{2\pi}\int_{-\pi}^{\pi}X(e^{j\omega})e^{j\omega n}d\omega$
- **帕斯瓦尔定理**：$\sum_{n=-\infty}^{+\infty}|x[n]|^2 = \frac{1}{2\pi}\int_{-\pi}^{\pi}|X(e^{j\omega})|^2 d\omega$

### 常用变换对速查
| 时域 | 频域 |
|------|------|
| $\delta(t)$ | $1$ |
| $1$ | $2\pi\delta(\omega)$ |
| $e^{-at}u(t)$ | $\frac{1}{a+j\omega}$ |
| $u(t)$ | $\pi\delta(\omega) + \frac{1}{j\omega}$ |
| $\cos(\omega_0 t)$ | $\pi[\delta(\omega-\omega_0) + \delta(\omega+\omega_0)]$ |
| $\sin(\omega_0 t)$ | $\frac{\pi}{j}[\delta(\omega-\omega_0) - \delta(\omega+\omega_0)]$ |
| $\frac{\sin(at)}{at}$ | $\frac{\pi}{a}[u(\omega+a) - u(\omega-a)]$ |
| 周期冲激串 $\sum\delta(t-nT)$ | $\frac{2\pi}{T}\sum\delta(\omega-k\frac{2\pi}{T})$ |

### LTI系统频域分析
- **频率响应**：$H(j\omega) = \frac{Y(j\omega)}{X(j\omega)}$
- **时域卷积**：$y(t) = x(t) * h(t) \leftrightarrow Y(j\omega) = X(j\omega)H(j\omega)$
- **微分方程**：$\sum a_k\frac{d^ky}{dt^k} = \sum b_k\frac{d^kx}{dt^k} \leftrightarrow H(j\omega) = \frac{\sum b_k(j\omega)^k}{\sum a_k(j\omega)^k}$
- **差分方程**：$\sum a_k y[n-k] = \sum b_k x[n-k] \leftrightarrow H(e^{j\omega}) = \frac{\sum b_k e^{-j\omega k}}{\sum a_k e^{-j\omega k}}$
