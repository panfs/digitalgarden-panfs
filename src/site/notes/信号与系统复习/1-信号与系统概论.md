## 一、信号与系统概论
### 1、信号的认识与分类
信号通常用函数和波形描述，一些信号特性可能无法用函数描述但可以有具体波形。
#### （1）随机信号与确定信号
**确定信号**：对给定的时间，其值是确定的不带有随机性。例如 $x(t) = \cos(\omega t)$。
**随机信号**：对给定的时间，其值是随机的或者不确定的。例如噪声信号。
#### （2）连续时间信号和离散时间信号

**连续时间信号**：用 $x(t)$ 描述，$t$ 表示连续时间变量，在连续时间范围内均有定义，其定义域是实数集。

$$x(t), \quad t \in \mathbb{R}$$

**离散时间信号**：用 $x[n]$ 描述，$n$ 表示离散时间变量，仅在部分离散点上有值的定义，其定义域是整数集或离散实数集。

$$x[n], \quad n \in \mathbb{Z}$$

**例题**：判断以下信号的类型
- $x(t) = e^{-t}\cos(2\pi t)$ 是连续时间信号
- $x[n] = \sin(\frac{n\pi}{4})$ 是离散时间信号

#### （3）周期信号和非周期信号

**连续周期信号**：
$$x(t) = x(t+mT), \quad m = 0, \pm1, \pm2, \ldots$$
其中 $T$ 为周期。
**离散周期信号**：
$$x[n] = x[n+mN], \quad m = 0, \pm1, \pm2, \ldots$$
其中 $N$ 为周期。
**基波周期**：满足上式的最小非零正值；当 $T$ 或 $N$ 趋于无穷时，周期信号变为非周期信号。
**多个连续信号合成的信号周期性要求**：各信号均有周期性，各信号周期的比值是有理数。

**离散信号的周期性要求**：周期是有理数有意义，若是分数则乘一定倍数变为正整数。

#### （4）实信号和复信号

**实信号**：在任何时间点上都有实数值的信号。声音、温度、电压等都可以用实数值来表示的信号都是实信号。

**复信号**：包含实部和虚部的信号，可以表示为实数和虚数的组合。正弦波就是一个典型的复信号，可以表示为振幅乘以复指数的形式。

**欧拉公式**：
$$e^{j\theta} = \cos\theta + j\sin\theta$$

则 $\cos\theta = \frac{e^{j\theta} + e^{-j\theta}}{2}$，$\sin\theta = \frac{e^{j\theta} - e^{-j\theta}}{2j}$

**变形得到两个重要三角式**：
$$\cos(\omega t) = \frac{e^{j\omega t} + e^{-j\omega t}}{2}$$
$$\sin(\omega t) = \frac{e^{j\omega t} - e^{-j\omega t}}{2j}$$

#### （5）模拟信号和数字信号

**模拟信号**：在连续时间范围内变化的信号。模拟信号可以采用模拟方式来表示，通常是通过无限多个连续的信号值来描述。例如，电压信号可以是一个连续变化的电压波形。

**数字信号**：在离散时间点上变化的信号。数字信号是通过采样和量化来离散化模拟信号得到的，通常以二进制形式表示。例如，数字音频信号是通过对模拟音频信号进行采样和量化得到的。

#### （6）能量信号和功率信号

**能量信号**：能量有限，功率为零，即 $E < \infty, P = 0$

能量定义：
$$E = \int_{-\infty}^{+\infty}|x(t)|^2 dt \quad \text{（连续时间）}$$
$$E = \sum_{n=-\infty}^{+\infty}|x[n]|^2 \quad \text{（离散时间）}$$

且一般 $\int_{-\infty}^{+\infty}|x(t)|dt < +\infty$ 在 $\mathbb{R}$ 收敛。

**功率信号**：功率有限，能量无穷，即 $P < \infty, E \to +\infty$

功率定义：
$$P = \lim_{T \to +\infty}\frac{1}{2T}\int_{-T}^{T}|x(t)|^2 dt \quad \text{（连续时间）}$$
$$P = \lim_{N \to +\infty}\frac{1}{2N+1}\sum_{n=-N}^{N}|x[n]|^2 \quad \text{（离散时间）}$$

#### （7）奇信号与偶信号

**奇信号**：
- 连续：$x(-t) = -x(t)$
- 离散：$x[-n] = -x[n]$

**偶信号**：
- 连续：$x(-t) = x(t)$
- 离散：$x[-n] = x[n]$

**任何信号都能分解成一个奇信号和一个偶信号的叠加**：
$$x(t) = x_e(t) + x_o(t)$$

其中奇信号 $x_o(t) = \frac{x(t) - x(-t)}{2}$，偶信号 $x_e(t) = \frac{x(t) + x(-t)}{2}$，离散信号的拆分同理。

#### 【练习题】

**题目1**：判断 $x(t) = \cos(3\pi t) + \sin(5\pi t)$ 的周期性。

**答案**：
- $\cos(3\pi t)$ 的周期 $T_1 = \frac{2\pi}{3\pi} = \frac{2}{3}$
- $\sin(5\pi t)$ 的周期 $T_2 = \frac{2\pi}{5\pi} = \frac{2}{5}$
- 周期之比 $\frac{T_1}{T_2} = \frac{2/3}{2/5} = \frac{5}{3}$ 是有理数
- 公共周期 $T = 3T_1 = 5T_2 = 2$

---

**题目2**：判断 $x[n] = \cos(\frac{3\pi n}{4})$ 的周期性。

**答案**：
- 数字频率 $\omega_0 = \frac{3\pi}{4}$
- 若周期为 $N$，则 $\frac{3\pi}{4}N = 2\pi k$，即 $N = \frac{8k}{3}$
- 取 $k=3$，得最小正整数周期 $N=8$

---

**题目3**：判断以下信号是能量信号还是功率信号
- $x(t) = e^{-2t}u(t)$
- $x(t) = \cos(\omega t)$

**答案**：
- 对于 $x(t) = e^{-2t}u(t)$：
  $$E = \int_{0}^{+\infty}e^{-4t}dt = \left[-\frac{1}{4}e^{-4t}\right]_0^{+\infty} = \frac{1}{4} < \infty$$
  是能量信号

- 对于 $x(t) = \cos(\omega t)$：
  $$E = \int_{-\infty}^{+\infty}\cos^2(\omega t)dt = +\infty$$
  $$P = \lim_{T \to +\infty}\frac{1}{2T}\int_{-T}^{T}\cos^2(\omega t)dt = \frac{1}{2} < \infty$$
  是功率信号

---

**题目4**：将 $x(t) = e^{-t}u(t)$ 分解为奇偶分量。

**答案**：
$$x_e(t) = \frac{e^{-t}u(t) + e^{t}u(-t)}{2}$$
$$x_o(t) = \frac{e^{-t}u(t) - e^{t}u(-t)}{2}$$

### 2、信号【自变量】的基本变换

#### （1）时移

:$x(t-t_0)$ 向右移动 $t_0$ 单位得到 $x(t-t_0)$，即整个图像向右平移 $t_0$ 单位。

:$x(t+t_0)$ 向左移动 $t_0$ 单位得到 $x(t+t_0)$，即整个图像向左平移 $t_0$ 单位。

**规则**：改变括号内时间只平移该信号的时域图像左右位置，左加右减。

#### （2）翻转反折

$$x(-t)$$

:$x(-t)$ 是 $x(t)$ 关于纵轴的翻转。

#### （3）尺度变换

$$x(at)$$

当 $a > 1$ 时，信号在时间轴上压缩；当 $0 < a < 1$ 时，信号在时间轴上扩展。

**综合变换的一般步骤**（以 $x(at-t_0)$ 为例）：
1. 时移：$x(t-t_0)$
2. 翻转：$x(-t-t_0)$（如果需要）
3. 尺度变换：$x(at-t_0)$

或：
1. 时移：$x(t-\frac{t_0}{a})$
2. 尺度变换：$x(at-t_0)$

#### 【练习题】

**题目5**：已知 $x(t) = e^{-t}u(t)$，求 $x(-t)$。

**答案**：
$$x(-t) = e^{t}u(-t)$$

---

**题目6**：已知 $x(t) = e^{-t}u(t)$，求 $x(2t-1)$。

**答案**：
方法一：先时移后尺度
- $x(t-1) = e^{-(t-1)}u(t-1)$
- $x(2t-1) = e^{-(2t-1)}u(2t-1) = e^{-2t+1}u(t-\frac{1}{2})$

方法二：先尺度后时移
- $x(2t) = e^{-2t}u(2t) = e^{-2t}u(t)$
- $x(2t-1) = e^{-2(t-\frac{1}{2})}u(t-\frac{1}{2}) = e^{-2t+1}u(t-\frac{1}{2})$

### 3、基本信号

#### （1）正弦信号

信号与系统中正弦与余弦信号都统称为正弦信号，即 $x(t) = A\cos(\omega t + \phi)$ 在连续时间上总是周期信号。

对于离散时间上的正弦信号 $x[n] = A\cos(\omega_0 n + \phi)$，仅当 $\frac{\omega_0}{2\pi}$ 为有理数时有周期性。

#### （2）指数信号

**连续时间实指数信号** $x(t) = e^{at}$ 通常是非周期性的。

**虚指数信号** $x(t) = e^{j\omega_0 t}$，周期 $T = \frac{2\pi}{|\omega_0|}$。

**一般复指数信号** $x(t) = e^{(a+j\omega_0)t}$：
$$x(t) = e^{at} \cdot e^{j\omega_0 t} = e^{at}(\cos\omega_0 t + j\sin\omega_0 t)$$

称为阻尼正弦震荡。

当 $a < 0$ 时为振幅指数衰减的正弦信号，当 $a > 0$ 时为振幅指数增长的正弦信号。

**离散时间实指数序列** $x[n] = a^n$，当 $a < 0$ 或 $a > 1$ 时非周期。

**离散时间复指数序列** $x[n] = r^n e^{j\omega_0 n} = (re^{j\omega_0})^n$，仅当基波频率 $\omega_0 = \frac{2\pi k}{N}$ 且 $k, N$ 为整数时有周期。

#### （3）抽样信号

抽样信号表示为 $\frac{\sin t}{t}$，为 $t$ 的偶函数，过零点 $t = n\pi$，$n = \pm1, \pm2, \ldots$

**性质**：
- $\frac{\sin 0}{0} = 1$
- $\lim_{t \to \pm\infty}\frac{\sin t}{t} = 0$
- $\int_{-\infty}^{+\infty}\frac{\sin t}{t}dt = \pi$
- $\int_0^{+\infty}\frac{|\sin t|}{t}dt = +\infty$

### 4、奇异信号

#### （1）单位脉冲/冲激信号

**离散的单位脉冲** $\delta[n]$：
- $n = 0$ 时 $\delta[0] = 1$
- $n \neq 0$ 时 $\delta[n] = 0$

**抽样性质**：
$$\sum_{n=-\infty}^{+\infty}x[n]\delta[n-n_0] = x[n_0]$$

且 $x[n]\delta[n-n_0] = x[n_0]\delta[n-n_0]$

**连续的单位冲激** $\delta(t)$：
- $t = 0$ 时 $\delta(0) = +\infty$
- $t \neq 0$ 时 $\delta(t) = 0$

**抽样性质**：
$$\int_{-\infty}^{+\infty}x(t)\delta(t-t_0)dt = x(t_0)$$

且 $x(t)\delta(t-t_0) = x(t_0)\delta(t-t_0)$

**偶对称性** $\delta(-t) = \delta(t)$

**尺度性质**：
$$\delta(at) = \frac{1}{|a|}\delta(t), \quad a \neq 0$$

**冲激函数的导数（偶阶导数偶对称，奇阶导数奇对称）**：
$$\delta^{(n)}(-t) = (-1)^n\delta^{(n)}(t)$$

**冲激函数与信号相乘的导数性质**：

对于任意光滑信号 $f(t)$：

$$f(t)\delta^{(n)}(t) = \sum_{k=0}^{n}(-1)^k \binom{n}{k} f^{(n-k)}(0)\delta^{(k)}(t)$$

特别地：
$$f(t)\delta'(t) = f(0)\delta'(t) - f'(0)\delta(t)$$
$$f(t)\delta''(t) = f(0)\delta''(t) - 2f'(0)\delta'(t) + f''(0)\delta(t)$$
$$f(t)\delta'''(t) = f(0)\delta'''(t) - 3f'(0)\delta''(t) + 3f''(0)\delta'(t) - f'''(0)\delta(t)$$

**冲激串（周期冲激序列）**：
$$\sum_{n=-\infty}^{+\infty}\delta(t - nT)$$

**冲激串的傅里叶变换**：
$$\mathcal{F}\left\{\sum_{n=-\infty}^{+\infty}\delta(t - nT)\right\} = \frac{2\pi}{T}\sum_{k=-\infty}^{+\infty}\delta(\omega - \frac{2\pi k}{T})$$

#### （2）单位阶跃信号

**离散** $u[n]$：
- $n < 0$ 时 $u[n] = 0$
- $n \geq 0$ 时 $u[n] = 1$

**连续** $u(t)$：
- $t < 0$ 时 $u(t) = 0$
- $t = 0$ 时函数无定义（或取 $u(0) = \frac{1}{2}$）
- $t > 0$ 时 $u(t) = 1$

**关系**：
- 离散时间单位阶跃是其单位冲激的求和函数：$u[n] = \sum_{k=-\infty}^{n}\delta[k]$
- 反之单位脉冲是离散时间单位阶跃的一次差分：$\delta[n] = u[n] - u[n-1]$

- 连续时间单位阶跃是其单位冲激的积分函数：$u(t) = \int_{-\infty}^{t}\delta(\tau)d\tau$
- 反之单位冲激是连续时间单位阶跃的一次微分：$\delta(t) = \frac{du(t)}{dt}$

#### （3）单位门函数信号

$$\Pi_T(t) = \begin{cases} 1, & |t| < \frac{T}{2} \\ \frac{1}{2}, & |t| = \frac{T}{2} \\ 0, & |t| > \frac{T}{2} \end{cases}$$

也可以表示为：$\Pi_T(t) = u(t+\frac{T}{2}) - u(t-\frac{T}{2})$

#### （4）单位符号函数信号

$$\text{sgn}(t) = \begin{cases} 1, & t > 0 \\ 0, & t = 0 \\ -1, & t < 0 \end{cases}$$

与阶跃函数关系：$\text{sgn}(t) = 2u(t) - 1$

**与冲激函数关系**：$\frac{d}{dt}\text{sgn}(t) = 2\delta(t)$

#### （5）奇异信号的其他重要性质

**冲激函数的筛选性质（推广）**：
$$\int_{-\infty}^{+\infty}x(t)\delta^{(n)}(t-t_0)dt = (-1)^n x^{(n)}(t_0)$$

特别地，对于一阶导数：
$$\int_{-\infty}^{+\infty}x(t)\delta'(t-t_0)dt = -x'(t_0)$$

**冲激函数与函数的卷积**：
$$x(t) * \delta(t) = x(t)$$
$$x(t) * \delta(t-t_0) = x(t-t_0)$$
$$x(t) * \delta'(t) = x'(t)$$
$$x(t) * \delta^{(n)}(t) = x^{(n)}(t)$$

**单位阶跃函数的积分性质**：
$$\int_{-\infty}^{t}u(\tau)d\tau = t \cdot u(t)$$

$$\int_{0}^{t}u(\tau)d\tau = t \cdot u(t)$$

**相关重要公式汇总**：

| 性质 | 公式 |
|------|------|
| 抽样性质 | $\int_{-\infty}^{+\infty}x(t)\delta(t-t_0)dt = x(t_0)$ |
| 乘法性质 | $x(t)\delta(t-t_0) = x(t_0)\delta(t-t_0)$ |
| 尺度性质 | $\delta(at) = \frac{1}{|a|}\delta(t)$ |
| 偶对称 | $\delta(-t) = \delta(t)$ |
| 导数对称 | $\delta^{(n)}(-t) = (-1)^n\delta^{(n)}(t)$ |
| 与导数乘积 | $f(t)\delta'(t) = f(0)\delta'(t) - f'(0)\delta(t)$ |
| 积分性质 | $\int_{-\infty}^{t}\delta(\tau)d\tau = u(t)$ |
| 卷积性质 | $x(t) * \delta(t-t_0) = x(t-t_0)$ |

#### 【练习题】

**题目7**：计算 $\int_{-\infty}^{+\infty}(t^2+3t+2)\delta(t-1)dt$

**答案**：
利用抽样性质：
$$\int_{-\infty}^{+\infty}(t^2+3t+2)\delta(t-1)dt = 1^2 + 3 \times 1 + 2 = 6$$

---

**题目8**：计算 $\int_{-\infty}^{+\infty}e^{-2t}\cos(3t)\delta'(t)dt$

**答案**：
利用冲激函数导数的筛选性质：
$$\int_{-\infty}^{+\infty}x(t)\delta'(t)dt = -x'(0)$$

设 $x(t) = e^{-2t}\cos(3t)$，则
$$x'(t) = -2e^{-2t}\cos(3t) - 3e^{-2t}\sin(3t)$$
$$x'(0) = -2 \times 1 \times 1 - 3 \times 1 \times 0 = -2$$

因此：
$$\int_{-\infty}^{+\infty}e^{-2t}\cos(3t)\delta'(t)dt = -(-2) = 2$$

---

**题目10**：计算 $(t+1)\delta(t-2) + 3\delta(t+1)$ 的表达式。

**答案**：
$$= (2+1)\delta(t-2) + 3\delta(t+1) = 3\delta(t-2) + 3\delta(t+1)$$

### 5、系统基本认识

#### （1）连续与离散

**连续时间系统**：输入和输出信号都是连续时间的，系统对信号的处理是在连续时间域进行的。如 $y(t) = \frac{dx(t)}{dt}$

**离散时间系统**：输入和输出信号都是离散时间的，系统对信号的处理是在离散时间域进行的。如 $y[n] = x[n] - x[n-1]$

#### （2）系统互联

**级联连接**：将多个系统连接在一起，其中一个系统的输出成为下一个系统的输入。这种连接方式类似于串联电路，信号从一个系统传递到另一个系统。

**并联连接**：将多个系统并排连接，其中相同的输入信号同时输入到每个系统，并将各系统的输出信号合并为最终的输出。这种连接方式类似于并联电路，各系统独立地处理相同的输入信号。

**反馈连接**：将系统的输出反馈到系统的输入，形成闭环系统。反馈连接可以是正反馈（增强系统的响应）或负反馈（稳定系统的响应）。

**馈送连接**：在系统之间引入额外的信号传递路径，以实现更复杂的功能。

#### （3）初始松弛

在信号与系统理论中，初始松弛（initial relaxation）是指系统在初始时刻对输入信号的响应，描述了系统在没有外部激励（输入信号）时的状态或者在系统稳定前的过渡状态。

- 连续时间系统中在 $t < 0$ 时，系统的输出 $y(t)$ 被假定为零
- 离散时间系统中在 $n < 0$ 时刻，离散时间系统的输出 $y[n]$ 也通常被假定为零

### 6、系统的基本性质

#### （1）线性与非线性

**线性系统**：同时满足叠加原理和齐次性质。

即系统的输出对于输入信号的加权和等于各个输入信号分别经过系统后的输出的加权和。

若 $x_1(t) \to y_1(t)$，$x_2(t) \to y_2(t)$，有：
$$ax_1(t) + bx_2(t) \to ay_1(t) + by_2(t)$$

**齐次性**：当输入 $x(t)$ 扩大 $k$ 倍，其输出也均匀扩大 $k$ 倍。
$$x(t) \to y(t) \Rightarrow kx(t) \to ky(t)$$

**叠加性**：若干个输入同时作用于系统时，其输出信号等于每个输入信号单独作用于系统产生的输出信号叠加。
$$x_1(t) \to y_1(t), x_2(t) \to y_2(t) \Rightarrow x_1(t) + x_2(t) \to y_1(t) + y_2(t)$$

**非线性系统**：不同时满足叠加原理和齐次性质。

#### （2）有记忆与无记忆

**无记忆系统**：对自变量的每一个值，一个系统的输出仅取决于该时刻输入。如 $y(t) = 2x(t)$，$y[n] = x^2[n]$，$x$ 与 $y$ 括号内的数值完全相同。

**有记忆系统**：对自变量的每一个值，一个系统的输出不仅取决于该时刻输入。如 $y(t) = x(t-1)$，$y[n] = \sum_{k=0}^{n}x[k]$ 这些。

#### （3）因果与非因果

**因果系统**：系统的输出只依赖于当前或过去的输入，即输入 $x$ 括号内的数不大于输出 $y$ 括号内的数。

对于任意 $t$（或 $n$），$y(t)$ 只取决于 $x(\tau)$ 其中 $\tau \leq t$（或 $x[k]$ 其中 $k \leq n$）。

对于LTI 系统， 如果$h[n] = 0, n < 0$ or $h(t) = 0, t<0$ 则系统是因果的。

**非因果系统**：系统的输出可能会依赖于未来的输入，即可能发生在输入信号之后。

#### （4）时变与时不变

**时变系统**：系统的性质随时间变化，即系统的输入输出关系在不同的时间段可能会发生变化。如果自变量发生了尺度变换或者幅度是关于时间的函数，那么系统就也是时变的。

**时不变系统**：系统的性质不随时间变化，即系统的输入输出关系在整个时间轴上都保持不变。

若 $x(t) \to y(t)$ 有 $x(t-t_0) \to y(t-t_0)$，或者幅度是常数，那么系统是时不变的。

#### （5）可逆性和可逆系统

**可逆性**：如果从系统的输出可以唯一地确定输入，那么这个系统就是可逆的。

**可逆系统**：能够在输出中包含足够信息以允许唯一确定输入的系统。这意味着系统的操作是可逆的，可以通过输出恢复出原始输入。

如 $y(t) = 2x(t)$ 的逆系统就是 $x(t) = \frac{1}{2}y(t)$。

常见的不可逆系统如 $y(t) = x^2(t)$ 和 $y(t) = \cos(x(t))$。

#### （6）稳定与不稳定

**稳定性**：当输入是有界时，输出也是有界的。

即若 $|x(t)| < M$（或 $|x[n]| < M$）时有 $|y(t)| < K$（或 $|y[n]| < K$）。

#### （7）线性相位系统

**定义**：系统的相位响应是频率的**线性函数**，即 $\phi(\omega) = -\alpha \omega$，可保证所有频率分量经历相同的时延，无相位失真。

**关键结论**：
- **并非所有 LTI 系统都是线性相位系统**
- 例如：一阶 RC 低通滤波器是 LTI 系统，但其相位响应是反正切函数 $\phi(\omega) = -\arctan(\omega RC)$，非线性
- 线性相位是系统传递函数的特定属性，与输入信号无关

**LTI 系统与线性相位系统的关系**：

| 特性 | LTI 系统 | 线性相位系统 |
|------|---------|-------------|
| 定义 | 满足线性和时不变性 | 相位响应是频率的线性函数 |
| 示例 | $y(t) = x(t-1)$，$y(t) = 2x(t) + x(t-2)$ | $y(t) = x(t-2)$ |
| 反例 | $y(t) = x(2t)$（时变），$y(t) = t \cdot x(t)$（时变） | 一阶 RC 低通滤波器（相位非线性） |


### 7、常用快速判断技巧（考试实用）

本节总结系统性质的快速判断方法，帮助在考试中快速识别系统的各种特性。

#### （1）LTI系统快速判断

**核心判断口诀**：
> 能用卷积表示的，都是LTI系统
> 输入输出满足齐次性和叠加性的，是线性系统
> 系数不随时间变化的，是时不变系统

**LTI系统特征**：

| 特征 | 说明 |
|------|------|
| **卷积表示** | $y(t) = x(t) * h(t)$ 或 $y[n] = x[n] * h[n]$ |
| **齐次性** | 若 $x(t) \to y(t)$，则 $ax(t) \to ay(t)$ |
| **叠加性** | 若 $x_1(t) \to y_1(t)$, $x_2(t) \to y_2(t)$，则 $x_1(t)+x_2(t) \to y_1(t)+y_2(t)$ |
| **时不变性** | 若 $x(t) \to y(t)$，则 $x(t-t_0) \to y(t-t_0)$ |

**示例识别**：
- $y(t) = x(t-2)$：✓ LTI系统（时移，系数不变）
- $y(t) = 2x(t) + x(t-1)$：✓ LTI系统（线性时不变，可用卷积表示）
- $y(t) = t \cdot x(t)$：✗ 非时不变（系数 $t$ 随时间变化）
- $y(t) = x^2(t)$：✗ 非线性（不满足叠加性）
- $y(t) = x(2t)$：✗ 非时不变（时间尺度变换）

#### （2）因果性快速判断

**核心判断口诀**：
> 输出不依赖未来输入的是因果系统
> 输出只依赖当前和过去输入的是因果系统

**判断方法**：

| 系统 | 判断依据 | 结论 |
|------|---------|------|
| $y(t) = x(t-2)$ | 输出 = 输入延迟2秒 | ✓ 因果 |
| $y(t) = x(t+1)$ | 输出 = 输入超前1秒 | ✗ 非因果 |
| $y[n] = x[n] + x[n-1]$ | 只依赖当前和过去 | ✓ 因果 |
| $y[n] = x[n+1]$ | 依赖未来 | ✗ 非因果 |
| $y(t) = \int_{-\infty}^{t}x(\tau)d\tau$ | 积分上限为 $t$ | ✓ 因果 |
| $y(t) = \int_{t}^{+\infty}x(\tau)d\tau$ | 积分上限为 $+\infty$ | ✗ 非因果 |

**记忆技巧**：
- 正延时（$t-\tau$, $t>\tau$）：因果
- 负延时（$t+\tau$, $t<\tau$）：非因果
- 积分上限为当前时间 $t$：因果
- 积分上限为无穷：非因果

#### （3）稳定性快速判断

**核心判断口诀**：
> 有界输入必有界输出是稳定系统
> BIBO稳定判断：$h(t)$ 或 $h[n]$ 绝对可积/可和

**判断方法**：

| 系统 | 判断依据 | 结论 |
|------|---------|------|
| $y(t) = e^{-at}u(t), a>0$ | $e^{-at}$ 绝对可积 | ✓ 稳定 |
| $y(t) = e^{at}u(t), a>0$ | $e^{at}$ 发散 | ✗ 不稳定 |
| $y[n] = a^n u[n], \|a\|<1$ | 等比级数收敛 | ✓ 稳定 |
| $y[n] = (-1)^n u[n]$ | 输出有界但不收敛 | ✓ 稳定（但有界） |
| $y(t) = t \cdot x(t)$ | 系数 $t$ 无界 | ✗ 不稳定 |
| $y[n] = \sum_{k=-\infty}^{n}x[k]$ | 累加可能无界 | ✗ 不稳定 |

**快速判断公式**：
- 连续系统：$\int_{-\infty}^{+\infty}|h(\tau)|d\tau < \infty$ → 稳定
- 离散系统：$\sum_{k=-\infty}^{+\infty}|h[k]| < \infty$ → 稳定

#### （4）可逆性快速判断

**核心判断口诀**：
> 能从输出唯一恢复输入的是可逆系统
> 输入不同时输出也不同的是可逆系统

**判断方法**：

| 系统 | 判断依据 | 结论 |
|------|---------|------|
| $y(t) = x(t-2)$ | 逆系统：$x(t) = y(t+2)$ | ✓ 可逆 |
| $y(t) = x(2t)$ | 逆系统：$x(t) = y(t/2)$ | ✓ 可逆 |
| $y(t) = x^2(t)$ | $x=2$ 和 $x=-2$ 输出相同 | ✗ 不可逆 |
| $y(t) = \|x(t)\|$ | $x=1$ 和 $x=-1$ 输出相同 | ✗ 不可逆 |
| $y(t) = e^{x(t)}$ | 单调递增，一一对应 | ✓ 可逆 |
| $y(t) = \cos(x(t))$ | 周期函数，不一一对应 | ✗ 不可逆 |

**记忆技巧**：
- 奇函数（如 $x^3, \sinh$）：通常可逆
- 偶函数（如 $x^2, |x|, \cos$）：通常不可逆
- 单调函数：可逆
- 周期函数：不可逆

#### （5）记忆性快速判断

**核心判断口诀**：
> 输出只依赖当前时刻的是无记忆系统
> 输出依赖过去时刻的是有记忆系统

**判断方法**：

| 系统 | 判断依据 | 结论 |
|------|---------|------|
| $y(t) = 2x(t)$ | 只依赖当前 | ✓ 无记忆 |
| $y[n] = x[n] + 1$ | 只依赖当前 | ✓ 无记忆 |
| $y(t) = x(t-1)$ | 依赖过去 | ✗ 有记忆 |
| $y[n] = x[n] + x[n-1]$ | 依赖过去 | ✗ 有记忆 |
| $y(t) = \int_{0}^{t}x(\tau)d\tau$ | 积分历史 | ✗ 有记忆 |
| $y(t) = \frac{dx(t)}{dt}$ | 微分涉及邻域 | ✗ 有记忆 |

#### （6）时不变性快速判断

**核心判断口诀**：
> 系数不显式包含时间变量 $t$ 的是时不变系统
> 输入时移等价于输出时移的是时不变系统

**判断方法**：

| 系统 | 判断依据 | 结论 |
|------|---------|------|
| $y(t) = 2x(t) + 3$ | 系数不含 $t$ | ✓ 时不变 |
| $y(t) = x(t-1) + x(t-2)$ | 只有输入时移 | ✓ 时不变 |
| $y(t) = t \cdot x(t)$ | 系数含 $t$ | ✗ 时变 |
| $y(t) = \cos(t) \cdot x(t)$ | 系数含 $t$ | ✗ 时变 |
| $y(t) = x(2t)$ | 时间尺度变换 | ✗ 时变 |
| $y[n] = n \cdot x[n]$ | 系数含 $n$ | ✗ 时变 |

**时变系统常见形式**：
- 系数是时间函数：$y(t) = a(t) \cdot x(t)$
- 时间尺度变换：$y(t) = x(at)$ 或 $y(t) = x(t/a)$
- 时间反转：$y(t) = x(-t)$

#### （7）综合性快速判断表

| 系统 | 线性 | 时不变 | 因果 | 稳定 | 可逆 | 记忆性 |
|------|------|--------|------|------|------|--------|
| $y(t) = x(t-2)$ | ✓ | ✓ | ✓ | ✓ | ✓ | ✗ |
| $y(t) = 2x(t) + 3$ | ✗ | ✓ | ✓ | ✓ | ✓ | ✓ |
| $y(t) = x^2(t)$ | ✗ | ✓ | ✓ | ✓ | ✗ | ✓ |
| $y(t) = t \cdot x(t)$ | ✓ | ✗ | ✓ | ✗ | ✓ | ✓ |
| $y[n] = x[n] + x[n-1]$ | ✓ | ✓ | ✓ | ✓ | ✗ | ✗ |
| $y[n] = (-1)^n u[n]$ | ✓ | ✗ | ✓ | ✓ | ✗ | ✓ |
| $y(t) = \int_{-\infty}^{t}x(\tau)d\tau$ | ✓ | ✓ | ✓ | ✗ | ✓ | ✗ |
| $y(t) = \frac{dx(t)}{dt}$ | ✓ | ✓ | ✓ | ✗ | ✗ | ✗ |

**考试应用技巧**：
1. **先看表达式形式**：是否为卷积形式（$y = x * h$）
2. **判断线性**：是否为 $ax + by \to ay + by$ 的形式
3. **判断时不变**：系数是否含时间变量
4. **判断因果**：是否有未来输入
5. **判断稳定性**：冲激响应是否绝对可积/可和
6. **判断可逆性**：是否为一一对应关系

#### 【练习题】

**题目11**：判断系统 $y(t) = t \cdot x(t)$ 是否线性。

**答案**：
设输入为 $ax_1(t) + bx_2(t)$
$$y = t \cdot [ax_1(t) + bx_2(t)] = a \cdot t x_1(t) + b \cdot t x_2(t) = ay_1(t) + by_2(t)$$
系统是线性的。

---

**题目12**：判断系统 $y(t) = x(t)^2$ 是否线性。

**答案**：
设输入为 $x_1(t) + x_2(t)$
$$y = [x_1(t) + x_2(t)]^2 = x_1(t)^2 + x_2(t)^2 + 2x_1(t)x_2(t) \neq x_1(t)^2 + x_2(t)^2$$
系统是非线性的。

---

**题目13**：判断以下系统是有记忆还是无记忆
- $y(t) = x(t) + x(t-1)$
- $y[n] = 3x[n]$

**答案**：
- $y(t) = x(t) + x(t-1)$ 是有记忆系统
- $y[n] = 3x[n]$ 是无记忆系统

---

**题目14**：判断以下系统的因果性
- $y(t) = x(t-2)$
- $y(t) = x(t+1)$
- $y[n] = x[n] + x[n+1]$

**答案**：
- $y(t) = x(t-2)$ 是因果系统
- $y(t) = x(t+1)$ 是非因果系统
- $y[n] = x[n] + x[n+1]$ 是非因果系统

---

**题目15**：判断系统 $y(t) = x(t) + x(t-1)$ 是否时不变。

**答案**：
设输入为 $x(t-t_0)$
$$y' = x(t-t_0) + x(t-t_0-1)$$
而 $y(t-t_0) = x(t-t_0) + x(t-t_0-1)$
由于 $y' = y(t-t_0)$，系统是时不变的。

---

**题目16**：判断系统 $y(t) = t \cdot x(t)$ 是否时不变。

**答案**：
设输入为 $x(t-t_0)$
$$y' = t \cdot x(t-t_0)$$
*只更改自变量*

而 $y(t-t_0) = (t-t_0) \cdot x(t-t_0)$
*整体延后输出*

由于 $y' \neq y(t-t_0)$（除非 $t_0 = 0$），系统是时变的。

---

**题目17**：判断系统 $y(t) = x(2t)$ 是否可逆。

**答案**：
构造逆系统 $x(t) = y(t/2)$，系统是可逆的。

---

**题目18**：判断系统 $y(t) = e^{t}x(t)$ 的稳定性。

**答案**：
若 $|x(t)| < M$，则
$$|y(t)| = |e^{t}x(t)| = e^{t}|x(t)| < Me^{t}$$
当 $t \to +\infty$ 时，$Me^{t} \to +\infty$，输出无界，系统不稳定。

---

**题目19**：判断系统 $y[n] = \frac{1}{2}x[n] + \frac{1}{2}x[n-1]$ 的稳定性。

**答案**：
若 $|x[n]| < M$，则
$$|y[n]| = \left|\frac{1}{2}x[n] + \frac{1}{2}x[n-1]\right| \leq \frac{1}{2}|x[n]| + \frac{1}{2}|x[n-1]| < M$$
系统稳定。

### 7、系统的描述
#### （1）输入输出描述法（微分、差分方程）
着眼于系统输入与输出的关系，适用于单输入-单输出系统。
**连续时间系统**：微分方程
$$\frac{d^ny(t)}{dt^n} + a_{n-1}\frac{d^{n-1}y(t)}{dt^{n-1}} + \cdots + a_0y(t) = b_m\frac{d^mx(t)}{dt^m} + \cdots + b_0x(t)$$
**离散时间系统**：差分方程
$$y[n] + a_{n-1}y[n-1] + \cdots + a_0y[n-N] = b_mx[n-M] + \cdots + b_0x[n]$$
#### （2）状态变量描述法（状态方程、输出方程）
既可以描述输入与输出的关系，还可以描述系统内部状态，可用于单输入-单输出系统和多输入-多输出系统。
**状态方程**：$\dot{x}(t) = Ax(t) + Bu(t)$（连续）
**输出方程**：$y(t) = Cx(t) + Du(t)$
#### （3）框图描述
框图描述是一种图形化表示信号处理系统结构和功能的方法，它将系统分解为各个组件，并用符号和箭头表示信号的流动和系统的操作。一个系统由若干个基本单元组成，包括加法器、数乘器、积分器、延时器等。
**基本单元**：
- 加法器：$\otimes$
- 数乘器：$\to a \to$
- 积分器：$\to \int dt \to$（连续）
- 延时器：$\to z^{-1} \to$（离散）
